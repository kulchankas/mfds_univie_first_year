# Analysis in Data Science — Exam Study Guide

A practice-first study guide for *Analysis in Data Science* (Ehler, *Mathematical
Structures of Computation*, ch. 11 & §7.5), organized into ten topic blocks tied to
the course's 13 problem sets.

**Live page:** open [`index.html`](index.html) in a browser, or enable GitHub Pages
(Settings → Pages → deploy from `main` / root) to serve it at
`https://kulchankas.github.io/analysis/`.

## Features

- Ten topic blocks (metric spaces → operator norms) with state / prove / compute framing
- Worked recipes, theorem/definition/pitfall boxes, and hand-drawn SVG figures
- Practice problems with hidden solutions, a 2.5-week plan, Julia template bank, and a one-page cheat sheet
- MathJax-rendered mathematics

## UX

- Reading-progress bar and scrollspy that highlights the current section
- Light / dark theme toggle (remembers your choice, respects system preference)
- Collapsible off-canvas navigation on mobile
- Copy-to-clipboard buttons on every code block
- Back-to-top button and a print / save-as-PDF stylesheet

Single self-contained `index.html` — no build step.
