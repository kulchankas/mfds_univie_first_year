Block 1Metric spaces, norms & topology

Covers PS1 · feeds everything later (norms reappear in operator-norm and convergence blocks). · 📖 Book Project pp. 11–23

What to know

Definition · Metric A metric on a set $X$ is $d:X\times X\to[0,\infty)$ with, for all $x,y,z$: (M1) $d(x,y)=0 \iff x=y$; (M2) symmetry $d(x,y)=d(y,x)$; (M3) triangle inequality $d(x,z)\le d(x,y)+d(y,z)$.
Definition · The $p$-metrics on $\mathbb R^n$ $\;d_p(x,y)=\Big(\sum_{k=1}^n |x_k-y_k|^p\Big)^{1/p}$ for $p\in[1,\infty)$, and $\;d_\infty(x,y)=\max_k |x_k-y_k|$. Balls: $B_\varepsilon(x)=\{y: d(x,y)\le \varepsilon\}$ (closed), $\mathring B_\varepsilon(x)=\{y:d(x,y)<\varepsilon\}$ (open).
Definition · Convergence & its speed (1.1.3, 1.1.7) $x_n\to x$ iff $\forall\varepsilon>0\,\exists N\,\forall n\ge N:\ d(x_n,x)\le\varepsilon$ (equivalently $d(x_n,x)\to 0$; $N=N(\varepsilon)$). Intuition: after some index $N$, the distance between the elements and the limit stays below any $\varepsilon>0$ you pick. The order of convergence: order $r>1$ if $d(x_{k+1},x)\le c\,d(x_k,x)^r$; linear if $d(x_{k+1},x)\le c\,d(x_k,x)$ with $c\in(0,1)$; sublinear if $\limsup_k d(x_k,x)^{1/k}=1$. Chain: order $r\Rightarrow$ linear $\Rightarrow\limsup\|x_k-x\|^{1/k}<1$. (This is the language reused for Newton in Block 8.)
Definition · Cauchy, complete, and why $\mathbb R,\mathbb C$ qualify (1.2.1–1.2.8) Cauchy: $\forall\varepsilon>0\,\exists N\,\forall n,m\ge N:\ d(x_n,x_m)\le\varepsilon$ — terms bunch together without naming a limit. Convergent $\Rightarrow$ Cauchy always; a space is complete when the converse holds. $\mathbb Q$ is not complete (the Heron iteration for $\sqrt2$ is Cauchy in $\mathbb Q$, limit $\notin\mathbb Q$); $\mathbb R$ is complete, and $\mathbb C$ inherits it via $|z|^2=z\bar z=(x+\mathrm iy)(x-\mathrm iy)=x^2+y^2$, so $|z_n-z_m|^2=|x_n-x_m|^2+|y_n-y_m|^2$ splits a complex Cauchy sequence into two real ones.

Solution logic for the recurring tasks

Prove "$d$ is a metric"

(M1) and (M2) are usually one line. The work is (M3). For $d_\infty$: for each coordinate $k$, $|x_k-z_k|\le|x_k-y_k|+|y_k-z_k|\le \max_j|x_j-y_j|+\max_j|y_j-z_j|$. The right side no longer depends on $k$, so take the max over $k$ on the left — that's the whole trick. For $d_p$ the triangle inequality is the Minkowski inequality; cite it (you are not expected to re-prove Minkowski unless asked).

Recipe · $d_\infty \le d_p \le n^{1/p}d_\infty$ and the limit (PS1 P2)
  1. Lower bound: one term dominates the sum, so $d_p^p=\sum_k|x_k-y_k|^p \ge \max_k|x_k-y_k|^p = d_\infty^p$. Take $p$-th roots.
  2. Upper bound: every term $\le d_\infty^p$, and there are $n$ of them, so $d_p^p\le n\,d_\infty^p$; take $p$-th roots to get $n^{1/p}d_\infty$.
  3. Limit: $n^{1/p}=e^{(\ln n)/p}\to 1$ as $p\to\infty$, so the squeeze forces $d_p\to d_\infty$.
PitfallThe squeeze argument needs $x,y$ fixed. You are showing pointwise convergence of the value, not uniform convergence of the function. Don't over-claim.
Practice · Block 1
  1. Show $d(x,y)=\dfrac{\|x-y\|_2}{1+\|x-y\|_2}$ is a metric on $\mathbb R^n$. (Hint: $t\mapsto t/(1+t)$ is increasing and subadditive.)
    Solution sketch
    M1, M2 immediate. For M3, let $\phi(t)=t/(1+t)$, increasing and concave with $\phi(0)=0$, hence subadditive: $\phi(a+b)\le\phi(a)+\phi(b)$. With $a=\|x-y\|_2,\ b=\|y-z\|_2$ and $\|x-z\|_2\le a+b$, monotonicity then subadditivity give $d(x,z)=\phi(\|x-z\|_2)\le\phi(a+b)\le\phi(a)+\phi(b)=d(x,y)+d(y,z)$.
  2. For $n=2$, sketch by hand the unit balls $B_1(0)$ under $d_1,d_2,d_\infty$ and order their areas. Then explain in one sentence why the $d_{10}$ ball looks "almost square."
    Solution sketch
    $d_1$ = diamond (area 2), $d_2$ = disk (area $\pi$), $d_\infty$ = square $[-1,1]^2$ (area 4). As $p\to\infty$ the ball swells toward the $d_\infty$ square because $n^{1/p}d_\infty\to d_\infty$; at $p=10$ it is nearly the full square with slightly rounded corners.
  3. On any set $X$, show the discrete metric $d(x,y)=1$ if $x\neq y$, else $0$, is a metric, and describe $\mathring B_{1/2}(x)$ and $\mathring B_2(x)$.
    Solution sketch
    M1–M2 clear; M3 holds because the RHS is $\ge 1$ whenever the LHS is $1$. $\mathring B_{1/2}(x)=\{x\}$ (only point within distance $<\tfrac12$); $\mathring B_2(x)=X$.

Completeness vs. compactness

These two are constantly mixed up. Both say "sequences behave well," but they answer different questions and have different strength.

💡 Intuition that clicksA space is compact if it is not only complete, but also finite in size (bounded).
The two definitions, side by side
  • Complete — every Cauchy sequence converges to a limit in the space. (“No gaps.”)
  • Compact — every sequence (Cauchy or not) has a subsequence converging to a limit in the space. (“No escape.”)
The quantifiers differ: completeness restricts to Cauchy sequences but demands the whole sequence converge; compactness applies to all sequences but only asks for a subsequence.
Key theorem · the crisp link A metric space is compact $\iff$ complete and totally bounded. Hence compact $\Rightarrow$ complete (Lemma 1.3.2), but complete $\not\Rightarrow$ compact — completeness is just missing the boundedness half. In $\mathbb R^n$, "totally bounded" $=$ "bounded," which is exactly Heine–Borel: $K\subseteq\mathbb R^n$ is compact $\iff$ closed and bounded.
SpaceComplete?Compact?Why
$[0,1]$✓✓closed & bounded in $\mathbb R$
$\mathbb R$✓✗complete, but $x_n=n$ has no convergent subsequence — unbounded
$(0,1)$✗✗$x_n=\tfrac1n$ is Cauchy with limit $0\notin(0,1)$
$\mathbb Q\cap[0,1]$✗✗bounded, but a sequence $\to$ an irrational escapes the set
closed unit ball of $\ell^2$✓✗complete & bounded, but not totally bounded ($\infty$-dim)
Pitfall"Closed and bounded $\Rightarrow$ compact" holds only in finite dimensions ($\mathbb R^n$, Heine–Borel). In an infinite-dimensional space the closed unit ball is closed and bounded yet not compact (the basis vectors $e_1,e_2,\dots$ satisfy $d(e_i,e_j)=\sqrt2$, so no subsequence is Cauchy). And $\mathbb R$ alone shows completeness never gives compactness on its own.

One-line memory: completeness = "limits that should exist, do" (no holes); compactness = completeness plus "nowhere to run" (totally bounded). Compactness is the stronger, more global property.

Handy · Truncating a metric (Lemma 1.3.12) $\tilde d(x,y)=\min\{1,d(x,y)\}$ is again a metric, and it leaves both convergence and compactness unchanged: $x_n\to x$ in $d\iff$ in $\tilde d$, and $K$ compact in $d\iff$ in $\tilde d$. So every metric space is equivalent to a bounded one — which is exactly why plain "bounded" is not a metric invariant, and why compactness needs total boundedness, not boundedness.

Block 2Continuity, openness & connectedness

Covers PS1 P3–P4 PS2 PS14 P2. · 📖 Book Project pp. 24–38

What to know

Theorem · Topological characterization of continuity $f:X\to Y$ is continuous $\iff$ $f^{-1}(U)$ is open in $X$ for every open $U\subseteq Y$.
Definitions you'll reuse Contraction: $\exists L<1$ with $d(f(x),f(y))\le L\,d(x,y)$. Path-connected: any two points are joined by a continuous path $\gamma:[0,1]\to X$. Banach fixed-point: a contraction on a complete space has a unique fixed point, reached by iteration $x_{k+1}=f(x_k)$.
Theorem · Continuity = preserving convergence (1.4.1–1.4.6) $f$ is continuous at $x$ iff $x_n\to x \Rightarrow f(x_n)\to f(x)$ (equivalent to the open-preimage version above). Compositions, sums and products of continuous maps are continuous, and $x\mapsto d(x,y_0)$ is continuous (1-Lipschitz) — it simply returns the distance from the fixed point $y_0$. Piecewise functions: at a junction point, check the limit from both sides — continuity fails if the one-sided limits disagree or differ from the value (e.g. $f(x)=4x^2+2$ for $x<0$ gives $\to 2\ne 1=f(0)$, so it is discontinuous at $0$).
Theorem · The compactness payoff — EVT (1.4.7–1.4.8) If $X$ is compact and $f$ continuous, then $f(X)$ is compact (continuous image of a compact set is compact). Hence a continuous $f:X\to\mathbb R$ on a compact $X$ attains its maximum and minimum — the Extreme Value Theorem. This is the reason compactness matters for optimization: it guarantees a minimizer exists.
Theorem · The connectedness payoff — IVT (1.4.10–1.4.14) Continuous images of path-connected sets are path-connected, and for $X\subseteq\mathbb R$: path-connected $\iff$ interval. Consequence — the Intermediate Value Theorem: continuous $f:[a,b]\to\mathbb R$ with $f(a)<0Why path-connectedness matters: (1) IVT-style reasoning — if $f$ is positive somewhere and negative somewhere on a connected set, it must vanish in between; (2) it tells you whether you can continuously move between two solutions.
Recipe · Bisection root-finding + error bound (1.4.15) For continuous $f$ with $f(a)>0,\ f(b)<0$: take the midpoint $x_n=\tfrac{a_n+b_n}{2}$, keep the half where the sign changes, repeat. The bracket halves each step, so $x_n\to$ a root $\hat x$ with error bound $|x_n-\hat x|\le \tfrac{1}{2^n}|b-a|$. Stop at a tolerance tol, but never below machine epsilon ($\approx$ 2*eps()) — the smallest gap the computer can resolve, below which a number is indistinguishable from zero.

myBisection (the exam's root-finder)

function myBisection(f,a,b; tol=1e-4, maxiter=50)
    x = Vector{Float64}(undef, maxiter)
    for k in 1:maxiter
        x[k] = (a+b)/2
        if abs(a-b) < tol || abs(f(x[k])) < 2*eps()  # stopping criteria
            return x[1:k]
        end
        if f(a)*f(x[k]) < 0
            b = x[k]
        else
            a = x[k]
        end
    end
    println("no convergence")
    return x
end

Theorem · Banach Fixed-Point Theorem, in full (1.4.17) The Banach fixed-point theorem finds a point that $f$ leaves unchanged ($f(x^*)=x^*$). A contraction shrinks distances: take two points, apply $f$ to both, and they end up closer than they started — $d(f(x),f(y))\le q\,d(x,y)$ with $q\in[0,1)$. On a complete space such an $f$ has a unique fixed point $x^*$, and from any $x_0$ the Picard iterates $x_{n+1}=f(x_n)$ converge linearly to it, with $$d(x_n,x^*)\ \le\ \frac{q^{\,n}}{1-q}\,d(x_1,x_0).$$ Reading the bound: $d(x_n,x^*)$ is the current error — the distance between the $n$-th approximation and the actual true fixed point; $d(x_1,x_0)$ is the size of the first step; and $q$ is the shrinking factor. Since $q^{\,n}\to 0$, the factor $\tfrac{q^{n}}{1-q}$ acts as a "speed controller" — every step pushes the error a little closer to zero. (Existence: the iterates are Cauchy; completeness supplies the limit.)
Where the quantitative bound comes from (the "I didn't get it" part) The bound is just a geometric series of step sizes. Each step shrinks by $q$: $d(x_{k+1},x_k)\le q\,d(x_k,x_{k-1})\le\cdots\le q^{\,k}\,d(x_1,x_0)$. The distance still to travel to the limit is at most the sum of all future steps: $$d(x_n,x^*)\le\sum_{k\ge n}d(x_{k+1},x_k)\le\sum_{k\ge n}q^{\,k}\,d(x_1,x_0)=\frac{q^{\,n}}{1-q}\,d(x_1,x_0).$$ So in words: take one step you actually measured, $d(x_1,x_0)$; multiply by $q^{\,n}$ (how much $n$ steps have shrunk things) and by $\tfrac{1}{1-q}$ (the sum of the whole remaining tail). Because it uses only $x_0,x_1$ and $q$, you can evaluate it before iterating — an a-priori bound — to decide how many steps guarantee a target accuracy.
Example · the computational cycle (CRTA) on $f(x)=\tfrac14 x^2$ (1.4.18) Start with $f_0:(0,1)\to(0,1)$, $x\mapsto\tfrac14 x^2$. C — Completion: extend to $f:[0,1]\to[0,1]$ (a complete space). R — Representation/iterate: $|f(x)-f(y)|=\tfrac14|x+y|\,|x-y|\le\tfrac12|x-y|$ so $q=\tfrac12$ is a contraction; from $x_0=1$ the iterates $x_{n+1}=\tfrac14x_n^2$ converge to the fixed point $x^*=0$ (solving $x=\tfrac14x^2$). T/A — Truncation & approximation: with $d(x_1,x_0)=|\tfrac14-1|=\tfrac34$ the bound gives $|x_n|\le \dfrac{(1/2)^n}{1-1/2}\cdot\tfrac34 = 2^{-(n+1)}\cdot 3$ — geometric decay to $0$. (CRTA = Completion · Representation · Truncation · Approximation, the book's recurring computational cycle.)
Theorem · Three equivalent faces of continuity + sign-preservation (1.4.1–1.4.4) $f$ continuous at $x$ $\iff$ (sequential) $x_n\to x\Rightarrow f(x_n)\to f(x)$ $\iff$ ($\varepsilon$–$\delta$) $\forall\varepsilon\,\exists\delta:\ d_X(x,x')<\delta\Rightarrow d_Y(f(x),f(x'))<\varepsilon$. Globally $\iff$ $f^{-1}(U)$ open for every open $U$ $\iff$ $f^{-1}(C)$ closed for every closed $C$ (complement). Sign-preservation (1.4.4): if $f$ is continuous and $f(x)>0$, then $f>0$ on a whole neighborhood of $x$.
Why continuity preserves structure (Remark 1.4.19 — the "why?") Each "preservation" is just continuity applied to how that structure is defined:
  • Convergence: continuity literally is "$x_n\to x\Rightarrow f(x_n)\to f(x)$" — limits are preserved by definition.
  • Compactness: compact = "every sequence has a convergent subsequence"; push that subsequence through $f$ (it keeps converging) ⇒ $f(X)$ compact.
  • Path-connectedness: a path $\gamma$ maps to the path $f\circ\gamma$, still continuous ⇒ the image stays path-connected.
  • Topology: "$f^{-1}(\text{open})$ is open" is the topological face of continuity.
That is the "why": continuity is exactly the property that makes approximate inputs give approximate outputs, which is why almost all computation is built on continuous maps.
The ($\varepsilon$–$\delta$) picture & why it gives continuity (Lemma 1.4.1) $(\varepsilon$–$\delta)$ says: for every output tolerance $\varepsilon$ there is an input tolerance $\delta$ with $f\big(\mathring B_\delta(x)\big)\subseteq \mathring B_\varepsilon(f(x))$ — "stay within $\delta$ of $x$ and the output stays within $\varepsilon$ of $f(x)$." Why ($\varepsilon$–$\delta$) $\Rightarrow$ (sequential) continuity: let $x_n\to x$ and fix $\varepsilon$; take the matching $\delta$. Since $x_n\to x$, eventually $d(x_n,x)<\delta$, so $d(f(x_n),f(x))<\varepsilon$. As $\varepsilon$ was arbitrary, $f(x_n)\to f(x)$.
All the ways to show (or disprove) continuity — in one place To prove continuous: (a) sequential $x_n\to x\Rightarrow f(x_n)\to f(x)$; (b) $\varepsilon$–$\delta$; (c) $f^{-1}(\text{open})$ is open; (d) $f^{-1}(\text{closed})$ is closed; (e) build from known-continuous pieces — sums, products, compositions, coordinate maps $x\mapsto x_i$, and $x\mapsto d(x,y_0)$.
To disprove: give one sequence $x_n\to x$ with $f(x_n)\not\to f(x)$, or one open $U$ whose preimage isn't open.
Worked disproof (Heaviside): $f(x)=0$ for $x<0$ and $f(x)=1$ for $x\ge0$. Pick the open set $U=(0.5,1.5)$. Then $f(x)\in U\iff f(x)=1\iff x\ge0$, so $f^{-1}(U)=[0,\infty)$ — not open (the point $0$ has no surrounding interval inside it). Hence $f$ is not continuous (the jump is at $0$).
Theorem · Uniform continuity (1.4.5–1.4.6) $f$ is uniformly continuous if one $\delta$ works for all points: $\forall\varepsilon\,\exists\delta\ \forall x_1,x_2:\ d_X(x_1,x_2)\le\delta\Rightarrow d_Y(f(x_1),f(x_2))\le\varepsilon$. Key result: a continuous function on a compact domain is automatically uniformly continuous (proof by extracting convergent subsequences from a would-be counterexample).
Plain vs uniform continuity — the difference, visualized The only difference is whether $\delta$ may depend on the point: plain continuity allows $\delta=\delta(x,\varepsilon)$ (a different $\delta$ at each $x$); uniform continuity demands one $\delta=\delta(\varepsilon)$ that works everywhere at once. It fails when the graph gets arbitrarily steep: e.g. $f(x)=x^2$ on $\mathbb R$ or $f(x)=1/x$ on $(0,1)$ are continuous but not uniformly continuous — the steeper it gets, the smaller a $\delta$ you need, and no single $\delta$ survives. On a compact domain steepness is bounded, so continuous $\Rightarrow$ uniformly continuous.
Theorem · Uniform convergence & the space $C_b(X,Y)$ (1.4.7–1.4.11) If $\sup_x d_Y(f_n(x),f(x))\to 0$ (uniform convergence), the limit of continuous $f_n$ is continuous — pointwise is not enough (e.g. $f_n(x)=x^n$ on $[0,1]$ has a discontinuous pointwise limit). The bounded continuous functions $C_b(X,Y)$ with $d_\infty(f,g)=\sup_x d_Y(f(x),g(x))$ form a metric space, and $C_b(X,Y)$ is complete whenever $Y$ is — the function-space completeness that powers Banach/Hilbert spaces in Block 10.
Pointwise vs uniform convergence — how to tell, and what it needs Pointwise: $f_n(x)\to f(x)$ for each fixed $x$ (the threshold $N$ may depend on $x$). Uniform: one $N$ works for all $x$ at once, i.e. $d_\infty(f_n,f)=\sup_x|f_n(x)-f(x)|\to 0$.
The test (do this): find the pointwise limit $f$, then compute $M_n=\sup_x|f_n(x)-f(x)|$. If $M_n\to0$ ⇒ uniform; if $M_n$ stays bounded away from $0$ ⇒ only pointwise.
Does $f_n$ have to be continuous? No — uniform convergence is defined for any functions (it is just $\sup$-distance $\to0$). Continuity of the $f_n$ is only needed for the payoff: "a uniform limit of continuous functions is continuous." (And for $d_\infty$ to be finite you work with bounded functions.) Shortcut consequence: if continuous $f_n$ converge to a discontinuous limit, the convergence cannot be uniform — it was only pointwise.
Example 1.4.9: $f_n(x)=x^n$ on $[0,1]$ → limit $0$ on $[0,1)$ and $1$ at $x=1$. The limit is discontinuous, and $M_n=\sup_{[0,1]}|x^n-f|\not\to0$ ⇒ not uniform. Example 1.4.10: $f_n=\mathbf 1_{[1/n,\,1]}$ → limit $1$ on $(0,1]$ and $0$ at $0$; again discontinuous ⇒ only pointwise.

Solution logic

Recipe · continuity ⇔ preimages open (PS1 P3) Translate $\varepsilon$–$\delta$ into balls. (⇒) Take $U$ open, $x\in f^{-1}(U)$; since $U$ open, $\mathring B_\varepsilon(f(x))\subseteq U$. Continuity gives $\delta$ with $f(\mathring B_\delta(x))\subseteq \mathring B_\varepsilon(f(x))\subseteq U$, so $\mathring B_\delta(x)\subseteq f^{-1}(U)$ — preimage is open. (⇐) Given $x,\varepsilon$, the set $\mathring B_\varepsilon(f(x))$ is open, so $f^{-1}$ of it is open and contains $x$; extract a $\delta$-ball inside it. Done.
Recipe · "$f(x)=d(x,y_0)$ is continuous; $B_\varepsilon$ closed; $\mathring B_\varepsilon$ open" (PS1 P4) Continuity of $x\mapsto d(x,y_0)$ follows from the reverse triangle inequality $|d(x,y_0)-d(x',y_0)|\le d(x,x')$ (it is 1-Lipschitz). Then $\mathring B_\varepsilon(x)=f^{-1}([0,\varepsilon))$ is the preimage of an open set ⇒ open; $B_\varepsilon(x)=f^{-1}([0,\varepsilon])$ is the preimage of a closed set ⇒ closed.
Recipe · coordinate maps & vector continuity (PS2 P2) $\chi_i(x)=x_i$ is continuous because $|\chi_i(x)-\chi_i(x')|=|x_i-x_i'|\le\|x-x'\|_2$ (take $\delta=\varepsilon$). Then $f=(f_1,f_2)$ continuous $\iff$ each $f_i=\chi_i\circ f$ continuous: forward by composition; backward because $\|f(t)-f(s)\|_2\le\sqrt{|f_1(t)-f_1(s)|^2+|f_2(t)-f_2(s)|^2}$ is small when each component is.
Recipe · contraction ⇒ unique fixed point (PS2 P4) For $f(x)=\tfrac14(x^2+1)$ on $[0,1]$: (i) maps into itself: $f$ increasing on $[0,1]$, $f(0)=\tfrac14,\ f(1)=\tfrac12$, so $f([0,1])\subseteq[\tfrac14,\tfrac12]\subset[0,1]$. (ii) contraction: $|f'(x)|=|x/2|\le\tfrac12<1$, so by MVT $L=\tfrac12$. (iii) Banach FPT ⇒ unique fixed point; iterate $x_{k+1}=f(x_k)$.

Block 3Curves & arc length

Covers PS3. · 📖 Book Project pp. 135–139

What to know

Core facts For $\gamma:[0,1]\to\mathbb R^n$ continuously differentiable: metric derivative $|\dot\gamma|(t)=\|\gamma'(t)\|_2$, and length $$\ell(\gamma)=\int_0^1 \|\gamma'(t)\|_2\,dt=\int_0^1\sqrt{\gamma_1'(t)^2+\cdots+\gamma_n'(t)^2}\,dt.$$ Length is independent of parametrization: if $\gamma=f\circ\phi$ with $\phi$ a $C^1$ bijection, $\ell$ is unchanged.

Solution logic

Recipe · compute a curve length (PS3 P1, P4) (1) Differentiate each component. (2) Form $\|\gamma'(t)\|_2=\sqrt{\sum_i \gamma_i'(t)^2}$ and simplify under the root first — most exam curves are rigged so the radicand collapses. (3) Integrate over $[0,1]$.

Worked (PS3 P1): $f(t)=\big(t,\ \tfrac{2}{3}t^{3/2}\big)$, so $f_1'=1$ and $f_2'=t^{1/2}$. Then $\|f'(t)\|_2=\sqrt{1+t}$, which integrates cleanly: $$\ell(f)=\int_0^1\sqrt{1+t}\,dt=\Big[\tfrac{2}{3}(1+t)^{3/2}\Big]_0^1=\tfrac{2}{3}\big(2\sqrt2-1\big)\approx1.219.$$ Note how the $t^{3/2}$ component was chosen precisely so the radicand becomes the linear $1+t$ — exam length problems are almost always rigged this way, so simplify under the root before integrating.
Recipe · reparametrization length (PS3 P4) For $\gamma=f\circ\phi$, $\phi(t)=\pi t^2$ on $[0,1]$ into $[0,\pi]$, $f(x)=(\cos x,\sin x)$: chain rule $\gamma'(t)=\phi'(t)f'(\phi(t))$, so $|\dot\gamma|(t)=|\phi'(t)|\,\|f'(\phi(t))\|=2\pi t\cdot 1=2\pi t$. Length $=\int_0^1 2\pi t\,dt=\pi$ — exactly half the unit circle, as it should be (upper semicircle).
Numerical companions (PS3 P2) Polygonal length: sum $\sum_j \|\gamma(t_j)-\gamma(t_{j-1})\|$ over a partition — converges up to $\ell(\gamma)$ from below. Midpoint rule: approximate $\int_0^1\|\gamma'\|$ by $\sum_j \|\gamma'(m_j)\|\,\Delta t$ at midpoints $m_j$. Compare both against the analytic value.

Block 4Differentiability, gradient & the affine-approximation pipeline

Covers PS4 PS5 · highest-frequency computation on the exam. · 📖 Book Project pp. 61–70, 125–134, 140–150

What to know

Derivative ↔ gradient ↔ Jacobian For $f:\mathbb R^n\to\mathbb R$ differentiable, the derivative is the row $f'(x)=\nabla f(x)^\top$ and $$f(x_0+h)=f(x_0)+\nabla f(x_0)^\top h + o(\|h\|).$$ For $f:\mathbb R^n\to\mathbb R^m$, the Jacobian $J_f(x)\in\mathbb R^{m\times n}$ has rows $\nabla f_i(x)^\top$, and the affine approximation is $f(x_0)+J_f(x_0)(x-x_0)$. Chain rule: $J_{g\circ f}(x)=J_g(f(x))\,J_f(x)$.
Fréchet derivative $f$ is Fréchet-differentiable at $x_0$ if a bounded linear map $A$ satisfies $\frac{\|f(x_0+h)-f(x_0)-Ah\|}{\|h\|}\to 0$. In $\mathbb R^n$ this $A$ is exactly $J_f(x_0)$.

The pipeline (memorize the motion, not formulas)

Recipe · gradient → affine approx → estimate (PS4 P1–P3, PS5 P4)
  1. Gradient: partial-differentiate componentwise. e.g. $f=x^2y+y^2\Rightarrow\nabla f=(2xy,\ x^2+2y)$.
  2. Evaluate at the base point $x_0$.
  3. Affine approx: $F(x)=f(x_0)+\nabla f(x_0)^\top(x-x_0)$ (scalar) or $f(x_0)+J_f(x_0)(x-x_0)$ (vector).
  4. Estimate a nearby value by plugging into $F$ (PS5 P4d: $f(1.01,0.98)$).
Recipe · least-squares derivative & normal equations (PS5 P2) $L(x)=\|Ax-b\|^2$. Expand $L=x^\top A^\top A x-2b^\top Ax+b^\top b$. Fréchet derivative: $L'(x)h=2(Ax-b)^\top A h$, i.e. $\nabla L(x)=2A^\top(Ax-b)$. Setting it to $0$ gives the normal equations $A^\top Ax=A^\top b$; convexity (Hessian $2A^\top A\succeq0$) makes any critical point a global minimum.
Recipe · chain rule by hand (PS5 P3) $f(x,y)=(x^2,y^2)^\top$, $g(u,v)=u-v$. $J_g=(1,-1)$, $J_f=\begin{pmatrix}2x&0\\0&2y\end{pmatrix}$, so $(g\circ f)'=(1,-1)\begin{pmatrix}2x&0\\0&2y\end{pmatrix}=(2x,\ -2y)$. Sanity check: $g\circ f=x^2-y^2$, gradient $(2x,-2y)$. ✓
PitfallGradient is a column, the derivative is a row. When you multiply $J_f(x_0)(x-x_0)$, dimensions must read $m\times n$ times $n\times1$. Mixing these up is the most common lost point on this block.

Block 5Taylor's formula, the Hessian & classifying extrema

Covers PS4 P5 PS6 PS7 P1–P2 PS8 P1–P2. · 📖 Book Project pp. 71–77, 89–96, 151–158

What to know

Theorem · Second-order Taylor in $\mathbb R^n$ $$f(x)=f(x_0)+\nabla f(x_0)^\top(x-x_0)+\tfrac12(x-x_0)^\top H_f(x_0)(x-x_0)+o(\|x-x_0\|^2),$$ where $H_f=\big(\partial_i\partial_j f\big)$ is the Hessian (symmetric when $f\in C^2$, by Schwarz/Clairaut).
Theorem · Second-order test At a critical point ($\nabla f=0$): $H_f\succ0$ ⇒ local min; $H_f\prec0$ ⇒ local max; $H_f$ indefinite ⇒ saddle; semidefinite/singular ⇒ test inconclusive.
The 2×2 shortcut For $H=\begin{pmatrix}a&b\\b&c\end{pmatrix}$: $\det H=ac-b^2$. If $\det H>0$ and $a>0$ → min; $\det H>0,\ a<0$ → max; $\det H<0$ → saddle; $\det H=0$ → inconclusive.

Solution logic

Recipe · find & classify all critical points (PS6 P3–P4, PS7 P2, PS8 P1–P2)
  1. Solve $\nabla f=0$ for all critical points (don't lose branches — factor carefully).
  2. Compute $H_f$ symbolically.
  3. Plug each critical point into $H_f$, apply the 2×2 shortcut (or eigenvalue signs in higher dim).
  4. If singular, fall back to direct inspection along lines through the point.
Worked (PS7 P2): $f=x^4+y^4-2x^2-2y^2$. $\nabla f=(4x^3-4x,\,4y^3-4y)=0\Rightarrow x,y\in\{0,\pm1\}$ → 9 critical points. $H_f=\operatorname{diag}(12x^2-4,\,12y^2-4)$. At $(0,0)$: $\operatorname{diag}(-4,-4)$ → max. At $(\pm1,\pm1)$: $\operatorname{diag}(8,8)$ → min. At mixed $(0,\pm1),(\pm1,0)$: signs differ → saddles.
Recipe · Taylor polynomial of given order (PS7 P1) For $f(x,y)=\log(1+x+y)$ at $(0,0)$: substitute $u=x+y$, use the 1-D series $\log(1+u)=u-\tfrac{u^2}{2}+\tfrac{u^3}{3}-\cdots$, then re-expand $u=x+y$. Order-3 polynomial $=(x+y)-\tfrac12(x+y)^2+\tfrac13(x+y)^3$. Evaluate at $(0.1,0.1)$: $u=0.2$ ⇒ $0.2-0.02+0.00267=0.18267$.
Why PS4 P5 worksWhen $f(x,y)=g(x+y)$, the multivariable Taylor of $f$ collapses to the 1-D Taylor of $g$ at $t=x+y$, because all mixed partials are equal ($\partial_x=\partial_y=g'$). Order-1 matches the affine approximation; order-2 with $g=e^t$ matches the Hessian form. This is a "recognize the structure" question, not a grind.

Block 6Convexity

Covers PS7 P3 · supports the optimization block. · 📖 Book Project pp. 76–78, 129–134

What to know

Definitions & characterizations $f$ convex: $f(\lambda x+(1-\lambda)y)\le\lambda f(x)+(1-\lambda)f(y)$ for $\lambda\in[0,1]$. First-order ($C^1$): $f(y)\ge f(x)+\nabla f(x)^\top(y-x)$. Second-order ($C^2$): $H_f\succeq0$ everywhere. Jensen: $f\big(\sum\lambda_i x_i\big)\le\sum\lambda_i f(x_i)$ for weights summing to 1.

Solution logic — the three closure rules (PS7 P3)

Recipe · prove these convex
  • Nonneg. combination $\sum\alpha_i f_i$, $\alpha_i\ge0$: add the convexity inequalities scaled by $\alpha_i$. Sum of convex is convex; scaling by $\ge0$ preserves it.
  • Pointwise max $g=\max_i f_i$: for each $i$, $f_i(\lambda x+(1-\lambda)y)\le\lambda f_i(x)+(1-\lambda)f_i(y)\le\lambda g(x)+(1-\lambda)g(y)$. Take max over $i$ on the left.
  • Affine precomposition $g(x)=f_1(Ax-b)$: convexity is preserved by affine maps because $A(\lambda x+(1-\lambda)y)-b=\lambda(Ax-b)+(1-\lambda)(Ay-b)$; feed into $f_1$'s convexity.
Exam tipThese three appear constantly. Each is a 3–4 line proof. Rehearse them until you can write them without thinking — they are free points.

    Solution sketch

Block 7Optimization algorithms: GD, SGD, mini-batch

Covers PS7 P4 PS8 P2 PS13 P4 · heavy Julia component. · 📖 Book Project pp. 158–165

What to know

Gradient descent & $L$-smoothness Iteration $x_{k+1}=x_k-\eta\nabla f(x_k)$. If $\nabla f$ is $L$-Lipschitz ($\|\nabla f(x)-\nabla f(y)\|\le L\|x-y\|$) and $f$ convex, GD with $\eta\le 1/L$ converges; for the quadratic case convergence is governed by the spectrum of the iteration matrix.
SGD & mini-batch for finite sums For $f=\tfrac1n\sum_i f_i$: SGD uses one $\nabla f_{i_k}$ per step; mini-batch averages over a batch $B_k$. Cheaper per step, noisier path, same minimizer in expectation.

Solution logic

Recipe · least-squares finite-sum gradient (PS7 P4) $f(x)=\tfrac1n\sum_i(a_i^\top x-b_i)^2$. Chain rule per term: $\nabla(a_i^\top x-b_i)^2=2(a_i^\top x-b_i)a_i$, so $\nabla f(x)=\tfrac{2}{n}\sum_i(a_i^\top x-b_i)a_i$. Then GD/SGD/MB differ only in which terms enter each step.
Recipe · GD as a linear map; stable step sizes (PS13 P4) For quadratic $f(x,y)=x^2+2xy+5y^2$: $\nabla f=2\begin{pmatrix}1&1\\1&5\end{pmatrix}u$. GD: $u_{k+1}=u_k-\tau\nabla f(u_k)=(I-2\tau Q)u_k=:A_\tau u_k$ with $Q=\begin{pmatrix}1&1\\1&5\end{pmatrix}$. $u_k\to0$ for all $u_0$ $\iff$ spectral radius $\rho(A_\tau)<1$ $\iff$ $|1-2\tau\mu|<1$ for both eigenvalues $\mu$ of $Q$ $\iff$ $0<\tau<1/\mu_{\max}$. Find eigenvalues of $Q$ ($\mu=3\pm\sqrt5$); stable range $0<\tau<\dfrac{1}{3+\sqrt5}$.
Recipe · Lipschitz gradient bound (PS8 P2b) For $f=\sin x+\cos y$, $\nabla f=(\cos x,-\sin y)$, $H_f=\operatorname{diag}(-\sin x,-\cos y)$ with entries bounded by 1, so $\|H_f\|\le1$ ⇒ $\nabla f$ is $1$-Lipschitz. In general $L=\sup\|H_f\|$ (operator norm of the Hessian).
Practice · Block 7
  1. For $f(x)=\tfrac12 x^\top Q x$ with $Q\succ0$, show GD converges for $0<\tau<2/\lambda_{\max}(Q)$.
    Solution sketch
    $u_{k+1}=(I-\tau Q)u_k$. Eigenvalues $1-\tau\lambda_i$; need $|1-\tau\lambda_i|<1$ for all $i$ ⇒ $0<\tau\lambda_i<2$ ⇒ $0<\tau<2/\lambda_{\max}$.
  2. Compute $L$ (Lipschitz constant of $\nabla f$) for $f(x,y)=\tfrac12(x^2+10y^2)$ and state the GD-stable step range.
    Solution sketch
    $H=\operatorname{diag}(1,10)$, $L=\|H\|=10$. Convergence for $0<\tau<2/10=0.2$; fastest at $\tau=2/(1+10)$.
  3. Explain in 3 sentences why SGD's loss curve $f(x_k)$ is non-monotone while full GD's is (for convex $f$, small $\eta$).
    Solution sketch
    Full GD uses the exact gradient, so with $\eta\le1/L$ each step decreases $f$ monotonically. SGD replaces the gradient by a single-sample estimate that is unbiased but noisy, so individual steps can increase $f$. On average the trajectory still descends, giving a jagged-but-decreasing curve.

Block 8Newton's method · implicit & inverse function theorems

Covers PS8 P3–P4 PS10 PS11 P1–P2. · 📖 Book Project pp. 166–172

What to know

Newton iterations 1-D: $x_{k+1}=x_k-\dfrac{f(x_k)}{f'(x_k)}$. $\mathbb R^n$: $x_{k+1}=x_k-J_f(x_k)^{-1}f(x_k)$, solved each step as a linear system $J_f(x_k)\,\Delta=-f(x_k)$ (in Julia: A\b). Converges quadratically near a simple root.
Implicit & Inverse Function Theorems IFT: if $F(a,b)=0$ and $\partial_y F(a,b)\neq0$, then locally $y=g(x)$ with $g'(x)=-\dfrac{\partial_x F}{\partial_y F}$. Inverse FT: if $J_f(x_0)$ is invertible, $f$ is locally a $C^1$ diffeomorphism with $J_{f^{-1}}=J_f^{-1}$.

Solution logic

Recipe · Newton as linearize-and-solve (PS8 P3) The affine approximation at $x_k$ is $F_k(x)=f(x_k)+f'(x_k)(x-x_k)$. Solving $F_k(x)=0$ gives $x=x_k-f(x_k)/f'(x_k)$ — exactly the Newton step. That's the whole justification: Newton replaces $f$ by its tangent and takes the tangent's root.
Recipe · Newton by hand in $\mathbb R^n$ (PS10 P1) For $f(x,y)=(x^2+y^2-1,\ x-y)$, $J_f=\begin{pmatrix}2x&2y\\1&-1\end{pmatrix}$. From $X_0$: compute $f(X_0)$, $J_f(X_0)$, solve $J_f\Delta=-f(X_0)$, set $X_1=X_0+\Delta$; repeat for $X_2$. Compare to the true roots $(\pm\tfrac{1}{\sqrt2},\pm\tfrac{1}{\sqrt2})$ to report the error.
Recipe · IFT to get $y=g(x)$ (PS11 P1) $f(x,y)=x^3+x-2y$. $\partial_y f=-2\neq0$ everywhere, so IFT applies at every zero: $y=g(x)$ exists locally. Here you can even solve explicitly: $y=\tfrac12(x^3+x)$, and $g'(x)=\tfrac12(3x^2+1)$ — matches $-\partial_x f/\partial_y f=-(3x^2+1)/(-2)$. ✓
Convergence orders (PS10 P4) Linear: $x_k=2^{-k}$ → $|x_{k+1}|/|x_k|=\tfrac12$ const. Quadratic: $x_k=2^{-2^k}$ → $|x_{k+1}|/|x_k|^2=1$. On a log-axis plot, linear is a straight line; quadratic curves sharply downward. Newton on a simple root is quadratic; on the gradient $\nabla f$ for $f=\|Ax-b\|^2$ (linear system) Newton converges in one step.
Pitfall (PS10 P3)If $J_f$ is singular at/near a point (e.g. where $\det J_f=0$), Newton's step blows up or stalls. Always note where $\det J_f=0$ before iterating.
Practice · Block 8
  1. Do two Newton steps by hand for $f(x)=x^2-2$ from $x_0=1$. How close to $\sqrt2$?
    Solution sketch
    $x_1=1-\frac{-1}{2}=1.5$; $x_2=1.5-\frac{0.25}{3}=1.41\overline{6}$. $\sqrt2\approx1.41421$, error $\approx0.0025$ — quadratic convergence visible.
  2. For $f(x,y)=(e^{x}\cos y,\ e^{x}\sin y)$ (PS11 P2), show $J_f$ is invertible everywhere but $f$ is not globally injective.
    Solution sketch
    $\det J_f=e^{2x}(\cos^2y+\sin^2y)=e^{2x}>0$ always → locally invertible (Inverse FT). Not globally injective: $f(x,y)=f(x,y+2\pi)$, since sin/cos are $2\pi$-periodic.
  3. Use IFT on $x^2+y^2=1$ near $(0,1)$ to get $y=g(x)$ and $g'(0)$.
    Solution sketch
    $F=x^2+y^2-1$, $\partial_yF=2y=2\neq0$ at $(0,1)$. $g'(x)=-\frac{2x}{2y}=-x/y$; $g'(0)=0$. (Explicitly $g(x)=\sqrt{1-x^2}$.)

Block 9Constrained extrema · submanifolds & Lagrange multipliers

Covers PS11 P3–P4 PS12 PS13 PS14 P1. · 📖 Book Project pp. 173–183

What to know

Submanifold & tangent/normal space $M=\{x:g(x)=0\}$, $g:\mathbb R^n\to\mathbb R^r$, is a $(n-r)$-dim submanifold if $J_g(x)$ has full rank $r$ on $M$. Tangent space $T_xM=\ker J_g(x)$; normal space $N_xM=\operatorname{span}\{\nabla g_1,\dots,\nabla g_r\}$.
Theorem · Lagrange multipliers At a constrained extremum, $\nabla f(x)=\sum_{i=1}^r\lambda_i\nabla g_i(x)$ together with $g(x)=0$. Solve this system for $(x,\lambda)$.

Solution logic

Recipe · the Lagrange multiplier method (PS12 P1,P3; PS13)
  1. Write the stationarity system $\nabla f=\sum\lambda_i\nabla g_i$ and the constraints $g_i=0$.
  2. Solve — usually by ratioing equations to eliminate $\lambda$, then using the constraint.
  3. Evaluate $f$ at every candidate; largest = max, smallest = min.
Worked (PS12 P1): $f=xyz$ on $x^2+y^2+z^2=12$. $\nabla f=(yz,xz,xy)=2\lambda(x,y,z)$. From symmetry $x^2=y^2=z^2=4$ ⇒ $x,y,z=\pm2$. $f=\pm8$. Max $8$ at even number of minus signs, min $-8$ otherwise.
Recipe · prove $M$ is a submanifold (PS11 P3–P4, PS13 P1) Compute $J_g$ and show it has full row rank everywhere on $M$. e.g. sphere $g=\|x\|^2-1$: $\nabla g=2x\neq0$ on $M$ → rank 1 → $(n-1)$-dim submanifold; $T_xM=\{v:x^\top v=0\}=x^\perp$, $N_xM=\operatorname{span}\{x\}$.
Recipe · Lagrangian Hessian / bordered system for classification (PS13 P3) Form $L(x,\lambda)=f(x)-\lambda^\top g(x)$. Stationary points solve $\nabla_x L=0,\ g=0$. Classify by the Hessian $H_xL$ restricted to the tangent space $T_xM$ (or via the bordered Hessian). For Newton on constraints, iterate on $F(x,\lambda)=\big(\nabla_xL,\ g\big)$ with Jacobian $\begin{pmatrix}H_xL&-J_g^\top\\J_g&0\end{pmatrix}$.
AM–GM via Lagrange (PS13 P1) Maximize $f=x_1\cdots x_n$ on $\sum x_i=n$, $x_i>0$. Stationarity gives $\nabla f=\lambda(1,\dots,1)$, i.e. $\prod_{j\neq i}x_j$ equal for all $i$ ⇒ all $x_i$ equal ⇒ $x_i=1$ ⇒ $f\le1$, equality iff all $x_i=1$. This is AM–GM.
Practice · Block 9
  1. Find the points of the ellipse $4x^2+y^2=16$ closest to and farthest from the origin (PS12 P2 variant).
    Solution sketch
    Minimize/maximize $x^2+y^2$ s.t. $4x^2+y^2=16$. $\nabla:(2x,2y)=\lambda(8x,2y)$. Either $x=0$ (⇒ $y=\pm4$, dist 4, farthest) or $y=0$ (⇒ $x=\pm2$, dist 2, closest).
  2. Minimize $f=x^2+y^2+z^2$ s.t. $x+y+z=6,\ x-y=2$ (PS12 P3).
    Solution sketch
    $\nabla f=\lambda(1,1,1)+\mu(1,-1,0)$ gives $2x=\lambda+\mu,\ 2y=\lambda-\mu,\ 2z=\lambda$. Substitute into $x+y+z=6$: $\tfrac{\lambda+\mu}{2}+\tfrac{\lambda-\mu}{2}+\tfrac{\lambda}{2}=\tfrac{3\lambda}{2}=6\Rightarrow\lambda=4$. And $x-y=\mu=2$. Hence $x=3,\ y=1,\ z=2$, the unique minimizer, with $f=9+1+4=14$.
  3. Show $M=\{x\in\mathbb R^3: x_1^2+x_2^2=1,\ x_3=x_1\}$ is a 1-dim submanifold (PS11 P4).
    Solution sketch
    $g=(x_1^2+x_2^2-1,\ x_3-x_1)$, $J_g=\begin{pmatrix}2x_1&2x_2&0\\-1&0&1\end{pmatrix}$. On $M$, $(x_1,x_2)\neq(0,0)$, and the two rows are independent (second has a nonzero in column 3 where the first is 0) → rank 2 → $3-2=1$-dim submanifold.

Block 10Operator norms & the (von) Neumann series

Covers PS9 PS14 P3 · textbook §7.5. · 📖 Book Project pp. 101–111

What to know

Operator norm For linear $A$, $\|A\|=\sup_{\|x\|=1}\|Ax\|$, depending on the chosen norms on domain/codomain. Sum-norm (1-norm) ⇒ column-sum: $\|A\|_1=\max_j\sum_i|a_{ij}|$. Max-norm (∞-norm) ⇒ row-sum: $\|A\|_\infty=\max_i\sum_j|a_{ij}|$.
Theorem · Neumann series In a Banach algebra $L(B)$, if $\|g\|<1$ then $\mathrm{id}-g$ is invertible and $(\mathrm{id}-g)^{-1}=\sum_{k=0}^\infty g^k$. Finite identity: $\sum_{k=0}^n g^k=(\mathrm{id}-g^{n+1})(\mathrm{id}-g)^{-1}$.

Solution logic

Recipe · prove $\|A\|_1=$ column-sum (PS9 P1a) Upper bound: $\|Ax\|_1=\sum_i|\sum_j a_{ij}x_j|\le\sum_j|x_j|\sum_i|a_{ij}|\le(\max_j\sum_i|a_{ij}|)\|x\|_1$. Attained: pick $x=e_{j^*}$ (unit vector at the maximizing column) → $\|Ae_{j^*}\|_1=\sum_i|a_{ij^*}|=$ the max. Equality both ways ⇒ the sup is the column-sum. The $\|A\|_\infty$ proof is the transpose argument with $x=\pm$ signs of the maximizing row.
Recipe · Neumann inverse by hand (PS9 P3a) To invert $A$, write $A=\mathrm{id}-g$ with $\|g\|<1$, then $A^{-1}=\sum_k g^k$. For a diagonal $A=\operatorname{diag}(\tfrac12,\tfrac13,\tfrac14)$: $g=\mathrm{id}-A=\operatorname{diag}(\tfrac12,\tfrac23,\tfrac34)$, and $A^{-1}=\sum_k g^k=\operatorname{diag}\big(\tfrac{1}{1-1/2},\tfrac{1}{1-2/3},\tfrac{1}{1-3/4}\big)=\operatorname{diag}(2,3,4)$. ✓
Recipe · induction for the partial-sum identity (PS9 P2b) Base $n=0$: $\mathrm{id}=(\mathrm{id}-g)(\mathrm{id}-g)^{-1}$ ✓. Step: assume $\sum_0^n g^k=(\mathrm{id}-g^{n+1})(\mathrm{id}-g)^{-1}$; add $g^{n+1}$ and factor to get $(\mathrm{id}-g^{n+2})(\mathrm{id}-g)^{-1}$. This is the matrix/operator version of the geometric-sum formula $\sum_0^n x^k=\frac{1-x^{n+1}}{1-x}$ — that is the connection PS9 P2a points at.
Recipe · a linear functional that is unbounded (PS9 P4) On $C([0,1])$ with $\|f\|=\int_0^1|f|$, the evaluation $S(f)=f(0)$ is linear but not continuous: take $f_n$ a tall thin spike at 0 with $\int|f_n|=\tfrac1n\to0$ but $f_n(0)=1$. So $S(f_n)\not\to0$ while $\|f_n\|\to0$ ⇒ unbounded. With the sup-norm $\|f\|_\infty$ instead, $|S(f)|\le\|f\|_\infty$, so $S$ becomes continuous.
Practice · Block 10
  1. Give matrices with $\|A\|_1=\|A\|_\infty$, $\|A\|_1>\|A\|_\infty$, $\|A\|_1<\|A\|_\infty$ (PS9 P1c).
    Solution sketch
    Equal: the identity $I$ (both $=1$), or any symmetric matrix. Greater ($\|A\|_1>\|A\|_\infty$): $A=\begin{pmatrix}3&0\\1&0\end{pmatrix}$ has max column sum $4$ but max row sum $3$. Less ($\|A\|_1<\|A\|_\infty$): the transpose $\begin{pmatrix}3&1\\0&0\end{pmatrix}$ has max column sum $3$, max row sum $4$. (Transposing swaps the two norms, which is the quick way to flip the inequality.)
  2. Use the Neumann series to approximate $(\mathrm{id}-g)^{-1}$ for $g=\begin{pmatrix}0&0.1\\0.2&0\end{pmatrix}$ to 2 terms and bound the error.
    Solution sketch
    $\|g\|_\infty=0.2<1$. $S_1=I+g=\begin{pmatrix}1&0.1\\0.2&1\end{pmatrix}$. Tail $\le\|g\|^2/(1-\|g\|)=0.04/0.8=0.05$ in $\infty$-norm.
  3. Prove: if $\|g\|<1$, the series $\sum g^k$ converges in operator norm. (Block-10 core.)
    Solution sketch
    $\|g^k\|\le\|g\|^k$ and $\sum\|g\|^k=\frac{1}{1-\|g\|}<\infty$. By completeness of $L(B)$ and absolute convergence (Weierstrass M-test analog), the partial sums form a Cauchy sequence, hence converge.

Newton 1-D with history (PS8 P4)

function myNewton(f, x0, tol, maxiter)
    df(x) = ForwardDiff.derivative(f, x)
    x = x0; iters = [x0]
    for k in 1:maxiter
        xn = x - f(x)/df(x); push!(iters, xn)
        abs(xn - x) < tol && return xn, k, iters
        x = xn
    end
    x, maxiter, iters
end

Newton in $\mathbb R^n$ (PS10)

function newtonND(f, x0; tol=1e-12, maxiter=50)
    x = copy(x0)
    for k in 1:maxiter
        J = ForwardDiff.jacobian(f, x); F = f(x)
        norm(F) < tol && return x, k
        x += J \ (-F)         # solve J Δ = -F
    end
    x, maxiter
end

Constrained Newton via Lagrangian (PS13)

function newtonConstrained(f, g, x0, λ0; tol=1e-10, maxiter=30)
    x = copy(x0); λ = copy(λ0); n = length(x); r = length(λ)
    for k in 1:maxiter
        ∇f = ForwardDiff.gradient(f, x)
        G  = ForwardDiff.jacobian(g, x)
        L(z) = f(z) - dot(λ, g(z))
        H  = ForwardDiff.hessian(L, x)
        F  = vcat(∇f - G'λ, g(x))
        norm(F) < tol && return x, λ, k
        K = [H -G'; G zeros(r,r)]
        Δ = K \ (-F)
        x += Δ[1:n]; λ += Δ[n+1:end]
    end
    x, λ, maxiter
end

  1. Show $d_\infty$ is a metric on $\mathbb R^n$, then prove $d_\infty\le d_p$.
    Answer

    Answer

  3. Length of $\gamma(t)=(t,\tfrac23 t^{3/2})$ on $[0,1]$.
    Answer
    $\|\gamma'\|=\sqrt{1+t}$, $\ell=\tfrac23(2\sqrt2-1)\approx1.219$.
  4. $f(x,y)=x^2y+y^2$. Give $\nabla f$ and the affine approximation at $(1,1)$.
    Answer
    $\nabla f=(2xy,\,x^2+2y)\to(2,3)$; $f(1,1)=2$. $F(x,y)=2+2(x-1)+3(y-1)$.
  5. $L(x)=\|Ax-b\|^2$. Derive $\nabla L$ and the minimizer condition.
    Answer
    $\nabla L=2A^\top(Ax-b)$; global min $\iff A^\top Ax=A^\top b$ (Hessian $2A^\top A\succeq0$ ⇒ convex).
  6. Classify all critical points of $f=x^4+y^4-2x^2-2y^2$.
    Answer

    Answer
