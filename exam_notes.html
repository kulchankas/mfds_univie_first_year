Part II · Definitions

Definitions to know cold

If you can recite each of these in 30 seconds without notes, the proofs become readable.

Foundation

Field— II.1.1

A set $\mathbb{F}$ with two distinct elements $0, 1 \in \mathbb{F}$ and two operations $\oplus, \otimes : \mathbb{F} \times \mathbb{F} \to \mathbb{F}$ satisfying:

  1. Commutativity: $a \oplus b = b \oplus a$ AND $a \otimes b = b \otimes a$
  2. Associativity: $(a \oplus b) \oplus c = a \oplus (b \oplus c)$ AND $(a \otimes b) \otimes c = a \otimes (b \otimes c)$
  3. Neutral elements: $0 \oplus a = a$ AND $1 \otimes a = a$
  4. Additive inverse: $\forall a \in \mathbb{F}, \exists b: a \oplus b = 0$
  5. Multiplicative inverse: $\forall a \in \mathbb{F} \setminus \{0\}, \exists b: a \otimes b = 1$
  6. Distributivity: $a \otimes (b \oplus c) = (a \otimes b) \oplus (a \otimes c)$

Examples: $\mathbb{Q}, \mathbb{R}, \mathbb{C}, \mathbb{F}_2, \mathbb{Z}_p$ for $p$ prime.

Function: injective, surjective, bijective
  • Injective: $x \neq x' \Rightarrow f(x) \neq f(x')$. Equivalently: $f(x) = f(x') \Rightarrow x = x'$.
  • Surjective: $f(X) = Y$. Every $y \in Y$ has a preimage.
  • Bijective: injective AND surjective. Equivalent to invertible.
Function composition

$(g \circ f)(x) = g(f(x))$. Apply $f$ first, then $g$. Associative: $(h \circ g) \circ f = h \circ (g \circ f)$.

Matrices

Matrix-vector product— II.4.1

For $A \in \mathbb{F}^{m \times n}$ and $x \in \mathbb{F}^n$:

$$(Ax)_i = \sum_{j=1}^n a_{ij} x_j$$

Equivalently: $Ax$ is a linear combination of the columns of $A$ with coefficients $x_1, \dots, x_n$.

Matrix product— II.4.3

For $A \in \mathbb{F}^{m \times r}$, $B \in \mathbb{F}^{r \times n}$:

$$(AB)_{ij} = \sum_{k=1}^r a_{ik} b_{kj}$$

See three ways to compute it.

Identity matrix

$I_n$ — diagonal entries all $1$, zero elsewhere. $I_n A = A$ and $A I_n = A$.

Inverse matrix— II.4.14

$A \in \mathbb{F}^{n \times n}$ is invertible iff there exists $B$ with $AB = I = BA$. Then $B = A^{-1}$ is unique.

Transpose

$(A^T)_{ij} = A_{ji}$. Key property: $(AB)^T = B^T A^T$ — order flips.

Triangular matrices
  • Lower-triangular: $a_{ij} = 0$ for $i < j$ (above diagonal is zero).
  • Unit lower-triangular: lower-triangular AND all diagonal entries $= 1$.
  • Strictly lower-triangular: lower-triangular AND all diagonal entries $= 0$.
  • Same definitions flip for upper-triangular.
Matrix rank— II.4.49

The smallest $r \in \mathbb{N}_0$ such that $A = \sum_{k=1}^r u_k v_k^T$. The rank of a matrix is the number of linearly independent rows (equivalently, columns). Equivalently (more useful on exam): the dimension of the column space.

Vector spaces

Vector space over $\mathbb{F}$

A set $V$ with $+: V \times V \to V$ and $\cdot : \mathbb{F} \times V \to V$ such that:

  1. $(V, +)$ is an abelian group: commutative, associative, has $0_V$, has additive inverses
  2. $1 \cdot v = v$
  3. $\alpha \cdot (\beta \cdot v) = (\alpha \beta) \cdot v$
  4. $(\alpha + \beta) \cdot v = \alpha v + \beta v$
  5. $\alpha \cdot (v + w) = \alpha v + \alpha w$
Subspace

$U \subseteq V$ is a subspace iff (1) $0 \in U$, (2) closed under $+$, (3) closed under scalar mult.

Combined: $\forall \lambda \in \mathbb{F}, u, u' \in U: \lambda u + u' \in U$, and $0 \in U$.

Span

The span of a set of vectors is the set of all linear combinations of those vectors. $\langle A \rangle$ = smallest subspace of $V$ containing $A$ = set of all finite linear combinations of elements of $A$:

$$\langle A \rangle = \left\{ \sum_{i=1}^n \lambda_i a_i : n \in \mathbb{N}, a_i \in A, \lambda_i \in \mathbb{F} \right\}$$
Linear independence

$A \subseteq V$ is linearly independent iff the only linear combination giving $0$ is the trivial one:

$$\sum_{i=1}^n \lambda_i v_i = 0 \quad \Rightarrow \quad \lambda_1 = \dots = \lambda_n = 0$$

for every finite subset $\{v_1, \dots, v_n\} \subseteq A$.

Spanning set / generating system

A spanning set (or generating system) for a vector space \(V\) is a set of vectors whose span is all of \(V\).

Formally, \(\{v_1, \ldots, v_k\}\) is a spanning set for \(V\) if:

\[ \text{span}\{v_1, \ldots, v_k\} = V \]$A \subseteq V$ with $\langle A \rangle = V$.

Basis

$\mathcal{B} \subseteq V$ that is BOTH linearly independent AND a spanning set.

Equivalent (III.1.38): minimal spanning set, OR maximal linearly independent set.

Dimension

$\dim V$ = the number of elements in any basis of $V$. Well-defined by Steinitz.

Memorize: $\dim \mathbb{F}^n = n$, $\dim \mathbb{F}_n[X] = n + 1$, $\dim M_{m \times n}(\mathbb{F}) = mn$.

Linear maps

Linear map

$f: V \to W$ is linear iff for all $v, v' \in V$, $\lambda \in \mathbb{F}$:

$$f(\lambda v + v') = \lambda f(v) + f(v')$$
Kernel, image, rank
  • Kernel: $\ker(f) = \{v \in V : f(v) = 0\} = f^{-1}(\{0\})$ — subspace of $V$.
  • Image: $\text{im}(f) = f(V) = \{f(v) : v \in V\}$ — subspace of $W$.
  • Rank: $\text{rk}(f) = \dim(\text{im}(f))$.

Key (III.2.11): $f$ injective $\iff \ker(f) = \{0\}$.

Sum, direct sum, complement
  • Sum: $W + \hat W = \{w + \hat w : w \in W, \hat w \in \hat W\}$.
  • Direct sum: $W \oplus \hat W$ — sum with $W \cap \hat W = \{0\}$. Every element has unique decomposition.
  • Complement: $W, \hat W$ complementary in $V$ iff $V = W \oplus \hat W$.

    • The whole point of "direct" is uniqueness. These two conditions are equivalent:

    • \( U \cap W = \{0\} \)
    • Every vector \( v \in U + W \) can be written as \( v = u + w \) in exactly one way (one \( u \in U \), one \( w \in W \)).
Part III · Computational recipes

The three procedures you'll use on Problems 3 and 4

Mechanical skills. Drill until automatic.

LU decomposition & substitution

The recipe for solving $Ax = b$ via LU is three explicit steps. Problem 3 always tests this, almost always with a 3×3 system, sometimes requiring partial pivoting.

Step 1 — Compute L and U

Row-reduce $A$ to upper-triangular form $U$. Each multiplier $\ell_{ij}$ you use to eliminate goes into $L$ at position $(i, j)$. $L$ has 1s on the diagonal. If a pivot would be zero, swap rows and track the permutation $P$ so that $PA = LU$.

Step 2 — Forward substitution: solve $Ly = b$

$L$ is lower-triangular with $L_{ii} = 1$, so each row has one new unknown. Go top-down:

$$y_1 = b_1, \quad y_i = b_i - \sum_{j=1}^{i-1} L_{ij} y_j \quad (i = 2, \dots, n)$$
Step 3 — Backward substitution: solve $Ux = y$

$U$ is upper-triangular, so each row has one new unknown counting from the bottom. Go bottom-up:

$$x_n = \frac{y_n}{U_{nn}}, \quad x_i = \frac{1}{U_{ii}} \left( y_i - \sum_{j=i+1}^{n} U_{ij} x_j \right) \quad (i = n-1, \dots, 1)$$
L before U, y before x. Forward (top-down) for L, backward (bottom-up) for U.

Worked 3×3 structure

For $A = LU$ with

$$L = \begin{pmatrix} 1 & 0 & 0 \\ \ell_{21} & 1 & 0 \\ \ell_{31} & \ell_{32} & 1 \end{pmatrix}, \quad U = \begin{pmatrix} u_{11} & u_{12} & u_{13} \\ 0 & u_{22} & u_{23} \\ 0 & 0 & u_{33} \end{pmatrix}$$

Forward $Ly = b$:

$$y_1 = b_1, \quad y_2 = b_2 - \ell_{21} y_1, \quad y_3 = b_3 - \ell_{31} y_1 - \ell_{32} y_2$$

Backward $Ux = y$:

$$x_3 = \tfrac{y_3}{u_{33}}, \quad x_2 = \tfrac{y_2 - u_{23} x_3}{u_{22}}, \quad x_1 = \tfrac{y_1 - u_{12} x_2 - u_{13} x_3}{u_{11}}$$
Submission format

Always present $L$, $U$, $y$, $x$ explicitly with labels. Kazeev & Schichl want to see the work — don't shortcut to the final answer.

Matrix inverse via row operations

For an invertible $A \in \mathbb{F}^{n \times n}$, to compute $A^{-1}$:

  1. Form the augmented matrix $[\,A \mid I\,]$.
  2. Row-reduce until the left half is $I$. Whatever appears on the right is $A^{-1}$.
  3. Result: $[\,I \mid A^{-1}\,]$.

Why it works: each row operation corresponds to left-multiplying by an elementary matrix $E$. If $E_k \cdots E_1 A = I$, then $E_k \cdots E_1 = A^{-1}$. Applying the same row ops to $I$ gives $E_k \cdots E_1 I = A^{-1}$.

Common pitfall

Annotate each row operation above the arrow. "$R_2 \to R_2 - 2R_1$" — Kazeev & Schichl grade on the steps, not just the answer.

Three ways to compute $AB$

One method is enough for the exam, but understand all three — they appear in proofs differently.

Entry-by-entry $(AB)_{ij} = \sum_k a_{ik} b_{kj}$ Default for raw computation

IV.1.4 — Dimension Theorem for Sums Tier 1 — 50% base rate
For finite-dimensional subspaces $W, \hat W$ of $V$: $\dim(W + \hat W) = \dim(W) + \dim(\hat W) - \dim(W \cap \hat W)$.
Q5(a) may ask you to define these first (2 pts)

This is the proof that pairs with the subspace-relations definitions (exam_02's actual Q5a):

Subspace
$U \subseteq V$ with $0 \in U$, closed under $+$ and under scalar multiplication.
Sum of two subspaces
$W + \hat W := \{ w + \hat w : w \in W,\ \hat w \in \hat W \}$ — the smallest subspace containing both.
Direct sum
The sum is direct, written $W \oplus \hat W$, iff $W \cap \hat W = \{0\}$. Equivalently: every element of the sum has a unique decomposition $w + \hat w$.
Complement
$\hat W$ is a complement of $W$ in $V$ iff $V = W \oplus \hat W$ (i.e. $W + \hat W = V$ and $W \cap \hat W = \{0\}$).

The construction

  1. Pick a basis $\{w_1, \dots, w_k\}$ of $W \cap \hat W$, where $k = \dim(W \cap \hat W)$.
  2. Extend to a basis $\{w_1, \dots, w_n\}$ of $W$, where $n = \dim(W)$. (Uses the basis extension corollary: every linearly independent set extends to a basis — Cor. III.1.48(ii).)
  3. Separately, extend $\{w_1, \dots, w_k\}$ to a basis $\{w_1, \dots, w_k, \hat w_{k+1}, \dots, \hat w_m\}$ of $\hat W$, where $m = \dim(\hat W)$.
  4. Claim: $\mathcal{B} := \{w_1, \dots, w_n, \hat w_{k+1}, \dots, \hat w_m\}$ is a basis of $W + \hat W$.

Size check: $|\mathcal{B}| = n + (m - k) = \dim W + \dim \hat W - \dim(W \cap \hat W)$. ✓

Proof that $\mathcal{B}$ spans $W + \hat W$

$W \subseteq \langle \mathcal{B} \rangle$ (since the basis of $W$ is inside $\mathcal{B}$), similarly $\hat W \subseteq \langle \mathcal{B} \rangle$, hence $W + \hat W \subseteq \langle \mathcal{B} \rangle$. Reverse direction trivial.

Proof that $\mathcal{B}$ is linearly independent — the hard part

Suppose $\sum_{i=1}^n \lambda_i w_i + \sum_{j=k+1}^m \mu_j \hat w_j = 0$. Rearrange:

$$\sum_{i=1}^n \lambda_i w_i = -\sum_{j=k+1}^m \mu_j \hat w_j$$

LHS $\in W$, RHS $\in \hat W$, so both lie in $W \cap \hat W$. The RHS is expressed in the basis of $\hat W$ with the $w_1, \dots, w_k$ coefficients all zero — but it's in $W \cap \hat W$, hence must be a combination of $w_1, \dots, w_k$ only. By linear independence of the basis of $\hat W$: all $\mu_j = 0$. Then $\sum \lambda_i w_i = 0$, and linear independence of the basis of $W$ gives all $\lambda_i = 0$. $\square$

IV.1.10 — Rank-Nullity Theorem Tier 1 — 33% base rate
For finite-dim $V, W$ and linear $f: V \to W$: $\dim(V) = \dim(\text{im}(f)) + \dim(\ker(f))$.
Q5(a) may ask you to define these first (2 pts)

Pairs with the kernel / image / rank definitions (same trio asked alongside Steinitz on exam_01):

Kernel
$\ker(f) := f^{-1}(\{0\}) = \{ v \in V : f(v) = 0 \}$ — a subspace of $V$.
Image
$\text{im}(f) := f(V) = \{ f(v) : v \in V \}$ — a subspace of $W$.
Rank
$\text{rk}(f) := \dim(\text{im}(f))$, the dimension of the image. (Nullity $= \dim(\ker f)$.)

Three moves to memorize

  1. Start with the kernel. Pick a basis $\{b_1, \dots, b_k\}$ of $\ker(f)$, so $k = \dim(\ker f)$.
  2. Extend to a basis of $V$. By the basis extension corollary (every linearly independent set extends to a basis — Cor. III.1.48(ii)), extend to $\{b_1, \dots, b_n\}$ where $n = \dim V$.
  3. Claim: $\{f(b_{k+1}), \dots, f(b_n)\}$ is a basis of $\text{im}(f)$. Has $n - k$ elements ⇒ $\dim(\text{im} f) = n - k$. ✓

Generating

The $b_1, \dots, b_n$ span $V$, so their images $f(b_1), \dots, f(b_n)$ span $\text{im}(f)$ — the image of a spanning set spans the image, by Prop III.2.11. But $f(b_1) = \dots = f(b_k) = 0$ (since $b_1, \dots, b_k \in \ker f$), so $f(b_{k+1}), \dots, f(b_n)$ alone generate $\text{im}(f)$.

Linearly independent

Suppose $\lambda_{k+1} f(b_{k+1}) + \dots + \lambda_n f(b_n) = 0$. By linearity, $f(\lambda_{k+1} b_{k+1} + \dots + \lambda_n b_n) = 0$, so $\lambda_{k+1} b_{k+1} + \dots + \lambda_n b_n \in \ker(f) = \langle b_1, \dots, b_k \rangle$. Hence there exist $\mu_1, \dots, \mu_k$ with

$$\lambda_{k+1} b_{k+1} + \dots + \lambda_n b_n = \mu_1 b_1 + \dots + \mu_k b_k.$$

Rearrange to $\mu_1 b_1 + \dots + \mu_k b_k - \lambda_{k+1} b_{k+1} - \dots - \lambda_n b_n = 0$. By linear independence of the basis $\{b_1, \dots, b_n\}$ of $V$, every coefficient vanishes: all $\mu_j = 0$ and all $\lambda_i = 0$. So the $f(b_{k+1}), \dots, f(b_n)$ are independent.

Conclusion (don't forget this line)

$\{f(b_{k+1}), \dots, f(b_n)\}$ spans $\text{im}(f)$ and is independent ⇒ it is a basis ⇒ $\dim(\text{im} f) = n - k$. Therefore

$$\dim(V) = n = k + (n-k) = \dim(\ker f) + \dim(\text{im} f). \qquad \square$$
Where the marks are

Name both citations explicitly: Cor. III.1.48(ii) for the basis extension, Prop III.2.11 for the image of a spanning set spanning the image. And always write the final count line — the curated source trails off after "all coefficients zero," but no closing equation = lost marks.

Full written version (reproduce this on paper)

Theorem. Let $V, W$ be finite-dimensional $\mathbb{F}$-vector spaces and $f: V \to W$ linear. Then $\dim(V) = \dim(\text{im} f) + \dim(\ker f)$.

Proof. Choose a basis $\{b_1, \dots, b_k\}$ of the subspace $\ker(f) \subseteq V$, so $\dim(\ker f) = k$. Since these vectors are linearly independent, the basis extension corollary — every linearly independent set extends to a basis (Cor. III.1.48(ii)) — lets us extend them to a basis $\{b_1, \dots, b_k, b_{k+1}, \dots, b_n\}$ of $V$, so $\dim(V) = n$. We claim $\{f(b_{k+1}), \dots, f(b_n)\}$ is a basis of $\text{im}(f)$; this proves the theorem.

Generating. The vectors $b_1, \dots, b_n$ span $V$, so their images $f(b_1), \dots, f(b_n)$ span $\text{im}(f)$ (the image of a spanning set spans the image, by Proposition III.2.11). By construction $b_1, \dots, b_k \in \ker(f)$, so $f(b_1) = \dots = f(b_k) = 0$. Dropping these, the remaining $f(b_{k+1}), \dots, f(b_n)$ still span $\text{im}(f)$.

Linearly independent. Suppose scalars $\lambda_{k+1}, \dots, \lambda_n \in \mathbb{F}$ satisfy

$$\lambda_{k+1} f(b_{k+1}) + \dots + \lambda_n f(b_n) = 0.$$

By linearity the left side is $f(\lambda_{k+1} b_{k+1} + \dots + \lambda_n b_n)$, so

$$\lambda_{k+1} b_{k+1} + \dots + \lambda_n b_n \in \ker(f).$$

Since $\{b_1, \dots, b_k\}$ is a basis of $\ker(f)$, there are scalars $\mu_1, \dots, \mu_k \in \mathbb{F}$ with

$$\lambda_{k+1} b_{k+1} + \dots + \lambda_n b_n = \mu_1 b_1 + \dots + \mu_k b_k,$$

that is,

$$\mu_1 b_1 + \dots + \mu_k b_k - \lambda_{k+1} b_{k+1} - \dots - \lambda_n b_n = 0.$$

But $\{b_1, \dots, b_n\}$ is a basis of $V$, hence linearly independent, so every coefficient vanishes: $\mu_1 = \dots = \mu_k = \lambda_{k+1} = \dots = \lambda_n = 0$. In particular all $\lambda_j = 0$, proving independence.

Conclusion. $\{f(b_{k+1}), \dots, f(b_n)\}$ spans $\text{im}(f)$ and is linearly independent, so it is a basis; hence $\dim(\text{im} f) = n - k$. Therefore $\dim(V) = n = k + (n-k) = \dim(\ker f) + \dim(\text{im} f)$. $\square$

III.1.47 — Steinitz Exchange Theorem Tier 1 — 17% base rate
Let $V$ have spanning set $\{v_1, \dots, v_n\}$. (i) Any linearly independent $A \subseteq V$ is finite with at most $n$ elements; if $A = \{a_1, \dots, a_k\}$ with $k < n$, we can renumber the $v_i$ so $\{a_1, \dots, a_k, v_{k+1}, \dots, v_n\}$ spans $V$. (ii) Any spanning set contains a finite spanning subset with at most $n$ elements.
Q5(a) may ask you to define these first (2 pts)

On exam_01, Steinitz was paired with the kernel / image / rank definitions. Be ready for those, plus the terms inside the theorem itself:

Kernel / Image / Rank
$\ker(f) = \{v : f(v)=0\}$; $\ \text{im}(f) = f(V)$; $\ \text{rk}(f) = \dim(\text{im} f)$.
Spanning (generating) set
$\{v_1, \dots, v_n\}$ spans $V$ iff every $v \in V$ is a linear combination $\lambda_1 v_1 + \dots + \lambda_n v_n$. Equivalently $\langle v_1, \dots, v_n \rangle = V$.
Linear independence
The only combination $\lambda_1 v_1 + \dots + \lambda_n v_n = 0$ is the trivial one (all $\lambda_i = 0$).
Basis
A set that is both linearly independent and spanning.
The idea in one sentence (read this first)

If you have a set that spans (big enough to build everything) and a set that's linearly independent (no redundancy), you can trade the independent vectors into the spanning set one at a time — each swap kicks out one spanning vector and keeps the spanning property intact. The payoff: an independent set can never be larger than a spanning set. That single inequality is what makes "dimension" a well-defined number.

How to read the proof

Everything rests on the Exchange Lemma (III.1.46): if $\{u_1, \dots, u_n\}$ spans $V$ and $w = \lambda_1 u_1 + \dots + \lambda_n u_n$ with $\lambda_1 \neq 0$, then $\{w, u_2, \dots, u_n\}$ still spans — a nonzero coefficient means $u_1$ became redundant once $w$ is available, so you swap $w$ in for $u_1$. Steinitz is just this lemma applied repeatedly, one $a_i$ at a time. The induction below tracks how many $a$'s you've swapped in so far.

Proof of part (i) — induction on $\ell$ = number of $a$'s swapped in

Base case $\ell = 1$: Pick $a_1 \in A$. Since $\{v_1, \dots, v_n\}$ spans, $a_1 = \sum \lambda_i v_i$. Since $A$ is linearly independent, $a_1 \neq 0$, so some $\lambda_i \neq 0$. Renumber the $v_i$ so $\lambda_1 \neq 0$. By the Exchange Lemma (III.1.46), $\{a_1, v_2, \dots, v_n\}$ spans $V$. ✓

Inductive step: Assume we have $\{a_1, \dots, a_\ell, v_{\ell+1}, \dots, v_n\}$ spanning $V$.

  • If $A = \{a_1, \dots, a_\ell\}$: done. ($|A| = \ell \leq n$.)
  • Otherwise, pick $a_{\ell+1} \in A \setminus \{a_1, \dots, a_\ell\}$. Write $$a_{\ell+1} = \sum_{i=1}^\ell \lambda_i a_i + \sum_{i=\ell+1}^n \lambda_i v_i$$

Key argument: Some $\lambda_i$ with $i > \ell$ must be nonzero. (If not, $a_{\ell+1} = \sum_{i=1}^\ell \lambda_i a_i$, giving $\sum_{i=1}^\ell \lambda_i a_i - a_{\ell+1} = 0$, a nontrivial relation — contradicting linear independence of $A$.) Renumber so $\lambda_{\ell+1} \neq 0$. By the Exchange Lemma, $\{a_1, \dots, a_{\ell+1}, v_{\ell+2}, \dots, v_n\}$ spans $V$. ✓

Termination: Either we exhaust $A$ first ($|A| \leq n$), or we reach $\ell = n$. In the second case, $\{a_1, \dots, a_n\}$ spans, so $\langle \{a_1, \dots, a_n\} \rangle = V = \langle A \rangle$. By III.1.36, $A = \{a_1, \dots, a_n\}$, so $|A| = n$. $\square$

Three moves to remember
  1. Always renumber the $v_i$ so the nonzero coefficient is at $v_{\ell+1}$.
  2. The key argument is: some $v$-coefficient must be nonzero, because otherwise $A$ would be dependent.
  3. Cite the Exchange Lemma (III.1.46) explicitly at each step.
Why it matters (the payoff)

Take any two bases $B_1, B_2$ of $V$. Each is both independent and spanning, so applying "independent $\leq$ spanning" both ways gives $|B_1| \leq |B_2|$ and $|B_2| \leq |B_1|$, hence $|B_1| = |B_2|$. That equality is the definition of dimension. Steinitz is the load-bearing wall under the entire theory of dimension — which is why it's a Tier-1 proof candidate even at only ~17% appearance frequency.

IV.1.15 — Rank of a Matrix Tier 2 — in scope, lower frequency
For $A \in M_{m,n}(\mathbb{F})$: (i) $\text{rk}(A)$ equals the dimension of the column span of $A$ in $\mathbb{F}^m$ (so $\text{rk}(A) = r$ iff $r$ columns are linearly independent but any $r+1$ are dependent). (ii) For invertible $C \in M_m$, $D \in M_n$: $\text{rk}(A) = \text{rk}(CA) = \text{rk}(AD)$. (iii) If $A$ is in row echelon form with $r$ pivots, then $\text{rk}(A) = r$.

Throughout, let $f: \mathbb{F}^n \to \mathbb{F}^m$, $f(v) = Av$, so the columns of $A$ are $f(e_1), \dots, f(e_n)$.

(i) — rank = column-span dimension

The columns $f(e_1), \dots, f(e_n)$ generate $\text{im}(f)$ (Prop IV.0.2), so the column span is $\text{im}(f)$ and its dimension is $\text{rk}(A) := \dim(\text{im} f)$. By Cor III.1.48 this generating set contains a basis, giving the "$r$ independent, any $r+1$ dependent" characterization.

(ii) — invariance under invertible multiplication

Let $g: \mathbb{F}^m \to \mathbb{F}^m$, $g(v) = Cv$, an isomorphism. Then $g$ restricts to a bijection $\text{im}(f) \to \text{im}(g \circ f)$, so $\text{rk}(CA) = \dim(\text{im}(g\circ f)) = \dim(\text{im} f) = \text{rk}(A)$. The same argument on the domain side gives $\text{rk}(AD) = \text{rk}(A)$.

(iii) — echelon form has rank = pivot count

Reduce $A$ to reduced row echelon form (same pivots) by left-multiplication by an invertible matrix (Thm II.6.9); by (ii) this preserves rank, so assume $A$ is reduced. The pivot columns $j_1, \dots, j_r$ are exactly $e_1, \dots, e_r$, hence $r$ independent columns and $\text{rk}(A) \ge r$. Conversely all rows from $r+1$ on are zero, so $\text{im}(f) \subseteq \mathbb{F}^r$, forcing $\text{rk}(A) \le r$. Thus $\text{rk}(A) = r$. $\square$

The ten smaller proofs

Anything from this list may appear inside Problem 2 or as part of Problem 5. Skeleton-level recall is enough.

II.5.7 — Triangular structure under matrix multiplication
Expand the product entry-by-entry and split the index sum — triangularity zeroes out everything except the surviving diagonal/below-diagonal terms.
Lower-tri × lower-tri = lower-tri with $(AB)_{ii} = A_{ii} B_{ii}$. Unit/strict properties preserved. Same for upper-tri.
  1. Fix $i \leq j$. Split $(AB)_{ij} = \sum_k A_{ik} B_{kj}$ into three ranges: $k < i$, $i \leq k \leq j$, $k > j$.
  2. Lower-triangularity kills the third sum ($A_{ik} = 0$ for $k > i$) and the first ($B_{kj} = 0$ for $k < j$).
  3. If $i < j$: middle sum has $B_{kj} = 0$ for $k = i$ through $j-1$, giving $0$. If $i = j$: only $k = i$ survives, giving $A_{ii} B_{ii}$.
  4. (b), (c) follow from (a) by checking diagonals. (d), (e), (f) by transposition: $(AB)^T = B^T A^T$.
II.5.8 — Inversion of a single Gaussian elimination step
Multiply the two candidates out — the cross-terms cancel and the quadratic term dies because $\ell_k = 0$, leaving exactly $I$.
For $\ell \in \mathbb{F}^m$ with first $k$ components zero and $e_k$ the $k$-th column of $I$: $(I - \ell e_k^T)^{-1} = I + \ell e_k^T$.
  1. Compute $(I \mp \ell e_k^T)(I \pm \ell e_k^T) = I \pm \ell e_k^T \mp \ell e_k^T - \ell(e_k^T \ell) e_k^T$.
  2. The cross-terms cancel. The quadratic term has $e_k^T \ell = \ell_k = 0$ by assumption.
  3. So both products $= I$. By def II.4.14, $I - \ell e_k^T$ is invertible with inverse $I + \ell e_k^T$. $\square$
II.5.9 — Representation of $r$ steps of Gaussian elimination
Induct on $r$ — each new factor's cross-terms vanish because the multiplier vectors have zeros in the already-processed positions, so the product is just the sum of the single steps.
For $\ell_1, \dots, \ell_r \in \mathbb{F}^m$ ($r \le m-1$) where $\ell_k$ has its first $k$ components zero: $\displaystyle (I + \ell_1 e_1^T)\cdots(I + \ell_r e_r^T) = I + \sum_{k=1}^r \ell_k e_k^T$.

Induction on $r$.

  1. Base ($r=1$): both sides equal $I + \ell_1 e_1^T$. Trivial.
  2. Step: assume the claim for $r=n$. Multiply both sides on the right by $I + \ell_{n+1} e_{n+1}^T$ and expand (associativity, identity, distributivity — II.4.10): $$\Big(I + \sum_{k=1}^n \ell_k e_k^T\Big)\big(I + \ell_{n+1}e_{n+1}^T\big) = I + \sum_{k=1}^{n+1} \ell_k e_k^T + \sum_{k=1}^n \ell_k (e_k^T \ell_{n+1}) e_{n+1}^T.$$
  3. The final sum vanishes: for each $k \le n$, $e_k^T \ell_{n+1}$ is component $k$ of $\ell_{n+1}$, which is zero by hypothesis. So the product equals $I + \sum_{k=1}^{n+1}\ell_k e_k^T$, completing the induction. $\square$
II.5.10 — Lower-triangular matrices and inversion
Build the inverse one column at a time by back-substitution; triangular structure means each step only depends on already-solved entries.
(a) Unit lower-tri ⇒ invertible with unit lower-tri inverse. (b) Lower-tri with no zero on diagonal ⇒ invertible with lower-tri inverse. (c) Invertible lower-tri ⇒ no zeros on diagonal.

(a)

Write $L - I = \sum_{k=1}^{m-1} \ell_k e_k^T$ where $\ell_k$ has first $k$ components zero. By Lemma II.5.9, $L = \prod_{k=1}^{m-1} (I + \ell_k e_k^T)$. Each factor invertible by II.5.8. So $L^{-1} = \prod_{k=m-1}^{1} (I - \ell_k e_k^T)$ (note reversed order). By II.5.7(b), $L^{-1}$ is unit lower-tri.

(b)

Let $D = \text{diag}(L)$. $D$ invertible (no zeros). $D^{-1} L$ is unit lower-tri ⇒ invertible by (a). So $L = D(D^{-1}L)$ is invertible. $L^{-1} = (D^{-1}L)^{-1} D^{-1}$, both lower-tri ⇒ $L^{-1}$ lower-tri by II.5.7(a).

(c) — by contradiction

Suppose $L$ invertible but $L_{kk} = 0$ for some smallest $k$.

  • If $k = 1$: row 1 of $L$ is zero ⇒ row 1 of $LL^{-1}$ is zero ≠ row 1 of $I$. Contradiction.
  • If $k = m$: column $m$ of $L$ is zero ⇒ column $m$ of $L^{-1}L$ is zero ≠ column $m$ of $I$. Contradiction.
  • If $1 < k < m$: construct invertible $X$ such that $XL$ has a zero row at position $k$. Then $XL$ is invertible, but row $k$ of $(XL)^{-1}(XL)$ is zero. Contradiction. $\square$
III.1.29 — The span is the smallest containing subspace
Show two inclusions: the span is itself a subspace containing the set, and any subspace containing the set must contain all its linear combinations — hence the span.
For $A \subseteq V$: (i) there is a unique subspace $\langle A \rangle$ containing $A$ such that every subspace $W \supseteq A$ has $\langle A \rangle \subseteq W$. (ii) $v \in \langle A \rangle$ iff $v$ is a finite linear combination of elements of $A$.

(i)

Existence: define $\langle A \rangle := \bigcap_{W \supseteq A, W \text{ subspace}} W$. Intersection of subspaces is a subspace; contains $A$; minimal by construction.

Uniqueness: if $\tilde W$ has the same property, $\tilde W \subseteq \langle A \rangle$ and $\langle A \rangle \subseteq \tilde W$ both follow from the property.

(ii)

($\Leftarrow$) Any linear combination of $A$ lies in every subspace containing $A$, hence in the intersection $\langle A \rangle$.

($\Rightarrow$) Define $W := \{0\} \cup \{\text{all finite linear combinations of } A\}$. $W$ is a subspace (closed under linear combinations directly). $A \subseteq W$, so $\langle A \rangle \subseteq W$ by (i). $\square$

III.1.36 — Three equivalent characterizations of linear independence
Prove a cycle of implications — "no nontrivial relation" ⇒ "unique representations" ⇒ "removing any vector shrinks the span" ⇒ back to the start.
For $A \subseteq V$: (i) $A$ linearly independent ⇔ (ii) every $v \in \langle A \rangle$ has a unique representation ⇔ (iii) $\langle B \rangle \subsetneq \langle A \rangle$ for every proper $B \subsetneq A$.

(i) ⇒ (ii)

If $v = \sum \lambda_i a_i = \sum \mu_i a_i$, then $\sum (\lambda_i - \mu_i) a_i = 0$ forces $\lambda_i = \mu_i$ by linear independence.

(ii) ⇒ (iii), contrapositive

If $\langle B \rangle = \langle A \rangle$ for $B \subsetneq A$, pick $v \in A \setminus B$. Then $v$ has two representations: as $1 \cdot v$, and as a combination of $B$ (not involving $v$). Contradicts uniqueness.

(iii) ⇒ (i), contrapositive

If $A$ linearly dependent, $\sum_{i=1}^n \lambda_i v_i = 0$ with $\lambda_1 \neq 0$. So $v_1 = \sum_{i=2}^n -\frac{\lambda_i}{\lambda_1} v_i$. Setting $B = A \setminus \{v_1\}$, every element of $\langle A \rangle$ is reachable from $B$, so $\langle B \rangle = \langle A \rangle$. $\square$

III.1.38 — Three equivalent characterizations of a basis
Tie "independent + spanning" to "maximal independent set" and "minimal spanning set" via a cycle of implications, using that adding/removing a vector breaks one property.
$\mathcal{B} \subseteq V$ is a basis ⇔ minimal spanning set ⇔ maximal linearly independent set.

(i) ⇔ (ii)

Spanning is in both. Minimality "$\langle A \rangle \neq V$ for $A \subsetneq \mathcal{B}$" is exactly III.1.36(iii), equivalent to linear independence.

(i) ⇒ (iii)

$\mathcal{B}$ l.i. Let $B \supsetneq \mathcal{B}$. Then $\langle B \rangle \supseteq \langle \mathcal{B} \rangle = V$. If $B$ were l.i., III.1.36(iii) gives $\langle \mathcal{B} \rangle \subsetneq \langle B \rangle = V$, contradicting $\langle \mathcal{B} \rangle = V$.

(iii) ⇒ (i)

$\mathcal{B}$ l.i. by hypothesis. For $v \in V$: if $v \in \mathcal{B}$ done; else $\mathcal{B} \cup \{v\}$ properly contains $\mathcal{B}$, so l.d. — argue $v$ appears with nonzero coefficient (else $\mathcal{B}$ would be dependent), solve for $v$ as combination of $\mathcal{B}$. $\square$

III.1.46 — Exchange Lemma
A nonzero coefficient lets you solve for the old vector in terms of the new one, so swapping it in keeps the span unchanged.
For $A \subseteq V$, $v \in \langle A \rangle$ with $v = \sum \lambda_i a_i$ and $\lambda_0 \neq 0$ on some $a_0 \in A$. Then $A' := (A \setminus \{a_0\}) \cup \{v\}$ satisfies $\langle A' \rangle = \langle A \rangle$.
  1. From $v = \lambda_0 a_0 + \sum_{i \neq 0} \lambda_i a_i$ with $\lambda_0 \neq 0$, solve for $a_0$: $$a_0 = \frac{1}{\lambda_0} v - \sum_{i \neq 0} \frac{\lambda_i}{\lambda_0} a_i$$
  2. So $a_0 \in \langle A' \rangle$, hence $A \subseteq \langle A' \rangle$ and $\langle A \rangle \subseteq \langle A' \rangle$.
  3. Conversely, $v \in \langle A \rangle$ and $A \setminus \{a_0\} \subseteq \langle A \rangle$, so $A' \subseteq \langle A \rangle$ and $\langle A' \rangle \subseteq \langle A \rangle$.
  4. $\langle A' \rangle = \langle A \rangle$. $\square$
III.1.47 — Steinitz Exchange Theorem
Apply the Exchange Lemma repeatedly, trading independent vectors into the spanning set one at a time — independent can never outnumber spanning.
The full proof lives in the big-proofs section above (it's a Tier-1 10-point candidate). See Steinitz Exchange Theorem. Quick reminders for part (ii) below.

(ii) — extracting a finite spanning subset (the part the big version compresses)

If $A = \{0\}$ take $A' = \emptyset$. Otherwise pick $a_1 \neq 0$; renumbering gives $\{a_1, v_2, \dots, v_n\}$ spanning. If $\langle\{a_1\}\rangle = V$, done; else there is $a_2 \in A \setminus \langle\{a_1\}\rangle$ (since $A$ spans), so some $v_i$ has a nonzero coefficient and exchanging gives $\{a_1, a_2, v_3, \dots, v_n\}$ spanning. Iterate: at step $\ell$, if $\{a_1, \dots, a_\ell\}$ doesn't span there is $a_{\ell+1} \in A \setminus \langle\{a_1, \dots, a_\ell\}\rangle$, forcing a nonzero $v$-coefficient; after $\le n$ exchanges the $a$'s alone span $V$, yielding $A'$. $\square$

III.2.11 — Kernel/image are subspaces; injectivity criterion
Use linearity to check the subspace axioms directly; injectivity ⇔ trivial kernel follows from $f(u)-f(v)=f(u-v)$.
For linear $f: V \to W$: (i) image of a subspace is a subspace of $W$; (ii) preimage of a subspace is a subspace of $V$; (iii) $f$ surjective ⇔ $\text{im}(f) = W$; (iv) $f$ injective ⇔ $\ker(f) = \{0\}$.

(i)

Let $w, w' \in f(\hat V)$. Then $w = f(v)$, $w' = f(v')$ for some $v, v' \in \hat V$. By linearity, $\lambda w + w' = f(\lambda v + v') \in f(\hat V)$. Also $f(0) = 0 \in f(\hat V)$. ✓

(ii)

Let $v, v' \in f^{-1}(\hat W)$. Then $f(v), f(v') \in \hat W$. $f(\lambda v + v') = \lambda f(v) + f(v') \in \hat W$, so $\lambda v + v' \in f^{-1}(\hat W)$. Also $f(0) = 0 \in \hat W$. ✓

(iv) ⇒

If $f$ injective and $f(v) = 0 = f(0)$, then $v = 0$.

(iv) ⇐

If $\ker f = \{0\}$ and $f(x) = f(y)$, then $f(x-y) = 0$, so $x - y = 0$, hence $x = y$. $\square$

Q2 small proofs

Associativity of function composition

For any $x$ in the domain of $f$:

$$((h \circ g) \circ f)(x) = (h \circ g)(f(x)) = h(g(f(x))) = h((g \circ f)(x)) = (h \circ (g \circ f))(x)$$

Equal at every point ⇒ functions equal. $\square$

Uniqueness of inverse function

Suppose $g, \tilde g$ both satisfy $g \circ f = \text{id}_X$ and $f \circ g = \text{id}_Y$. Then:

$$\tilde g = \tilde g \circ \text{id}_Y = \tilde g \circ (f \circ g) = (\tilde g \circ f) \circ g = \text{id}_X \circ g = g$$

Using associativity of composition in the middle step. $\square$

Part VI · Concept guides

The four pillars

If these are clear, every Problem 5 and Problem 6 collapses. If they're fuzzy, no amount of memorization saves you.

Subspace

A subset of a vector space that is itself a vector space using the same operations.

The 3-step check:

  1. $0 \in U$
  2. $u, u' \in U \Rightarrow u + u' \in U$
  3. $\lambda \in \mathbb{F}, u \in U \Rightarrow \lambda u \in U$

In $\mathbb{R}^3$, subspaces are exactly: $\{0\}$, lines through origin, planes through origin, all of $\mathbb{R}^3$. Anything that doesn't contain $0$, or curves, or doesn't extend infinitely is not a subspace.

The kernel trick — your most-used shortcut

If $U$ is defined by a linear condition ("$\phi(v) = 0$" for linear $\phi$), then $U = \ker(\phi)$ is automatically a subspace. Cite Prop III.2.11(ii) and you're done in one line.

Examples: $\{p : p(0) = 0\}$ is $\ker(\text{eval at 0})$. $\{v : h(v) = v\}$ is $\ker(h - \text{id})$. $\{p : \int_0^1 p = 0\}$ is $\ker(\text{integration})$.

Why the kernel is always a subspace

For any linear $f: V \to W$, $\ker f = \{v \in V : f(v) = 0\}$ is a subspace of $V$. The three checks all run on linearity:

  1. Zero: $f(0) = 0$ (true for every linear map), so $0 \in \ker f$.
  2. Addition: $u, v \in \ker f \Rightarrow f(u + v) = f(u) + f(v) = 0 + 0 = 0$.
  3. Scalar mult: $v \in \ker f,\ \lambda \in \mathbb{F} \Rightarrow f(\lambda v) = \lambda f(v) = \lambda \cdot 0 = 0$.

The deep reason: $\{0\}$ is a subspace of $W$, and the preimage of any subspace under a linear map is a subspace (Prop III.2.11(ii)). The kernel is just the preimage of $\{0\}$ — so it's a subspace for free, no axiom-checking needed. Linearity is the engine: it lets $f$ pass through sums and scalar multiples.

Fixed points and $\ker(h - \mathrm{id})$

$\mathrm{id}$ is the identity map, $\mathrm{id}(v) = v$. For linear $h: V \to V$, the map $h - \mathrm{id}$ sends $v \mapsto h(v) - v$. Its kernel is

$$\ker(h - \mathrm{id}) = \{v : h(v) - v = 0\} = \{v : h(v) = v\}$$

— precisely the vectors $h$ leaves fixed. So "the fixed-point set of $h$" and "$\ker(h - \mathrm{id})$" name the same set. This is why the fixed-point set is automatically a subspace (Problem 6 favorite): rather than checking three axioms by hand, just write "it equals $\ker(h - \mathrm{id})$, the kernel of a linear map, hence a subspace by III.2.11." One line, full marks.

Linearity, additivity, homogeneity

A map $f: V \to W$ is linear iff it satisfies both:

  • Additivity: $f(u + v) = f(u) + f(v)$ for all $u, v \in V$.
  • Homogeneity: $f(\lambda v) = \lambda f(v)$ for all $v \in V$, $\lambda \in \mathbb{F}$.

Equivalently, one combined condition: $f(\lambda u + \mu v) = \lambda f(u) + \mu f(v)$. A consequence (not a separate axiom) is $f(0) = 0$ — useful as a quick disqualifier: if $f(0) \neq 0$, the map is not linear.

Using these on Problem 6 ("is this map linear?")

To prove not linear, break one property with a concrete counterexample — homogeneity is usually easiest. A squared term like $v_1^2$ fails it: with $v = (1,0,\dots)$ and $\lambda = 2$, the offending component gives $f(2v) = 4 \neq 2 = 2f(v)$. A nonzero constant term fails via $f(0) \neq 0$. To prove linear, write $f(v) = Av$ for a matrix $A$ and note matrix multiplication is linear.

Which standard operations are linear (use these to spot subspaces)

Each of these is a linear map, so any condition built from them — set equal to $0$ — defines a subspace (a kernel):

  • Differentiation $D: p \mapsto p'$. Additivity $(p+q)' = p' + q'$ and homogeneity $(\lambda p)' = \lambda p'$ are the basic derivative rules. On coefficients, $a_0 + a_1 X + a_2 X^2 + a_3 X^3 \mapsto a_1 + 2a_2 X + 3a_3 X^2$ — each coefficient just scaled and shifted, plainly linear. (Higher derivatives $p''$, $p'''$ too, as repeated compositions.)
  • Evaluation at a point $E_c: p \mapsto p(c)$. $(p+q)(c) = p(c) + q(c)$ and $(\lambda p)(c) = \lambda\,p(c)$. Linear for any fixed $c$ — this is why $p(0)=0$, $p(1)=0$, etc. give subspaces.
  • Integration $p \mapsto \int_a^b p$. Linear by the linearity of the integral.

Compositions and combinations stay linear. So $p \mapsto p'(2)$ is "differentiate, then evaluate at 2" — a composition $E_2 \circ D$ of two linear maps, hence linear. Likewise $p \mapsto p''(0) + p(1)$ is a sum of linear maps, still linear. That's why exotic-looking conditions like $\{p : p'(2) = 0\}$ or $\{p : p''(0) + p(1) = 0\}$ are all subspaces — each is "[a linear map of $p$] $= 0$", i.e. a kernel.

What breaks linearity

A nonlinear operation on $p$: a product or power of evaluations like $p(2)^2 = 0$ or $p(0)\cdot p(1) = 0$. These are not linear in $p$, so they don't automatically give subspaces — check by hand (they usually fail, or collapse to something like $\{p : p(2)=0\}$).

Span (= linear hull)

Terminology — Kazeev & Schichl use several names for the same thing

The span $\langle A \rangle$ is also called the linear hull (or just the hull) of $A$ — this is their primary name for it (Def. II.5.20 / III.1.x). All of these mean the identical object:

  • span of $A$ = linear hull of $A$ = hull of $A$ = $\langle A \rangle$.
  • "$A$ generates $W$" / "$A$ is a generating system for $W$" means $\langle A \rangle = W$ — same as "$A$ spans $W$" / "$A$ is a spanning set."
  • A space is finitely generated iff it has a finite spanning set (equivalently, a finite linear hull generator) — i.e. finite-dimensional.

If the exam says "compute the linear hull" or "is $A$ a generating system," it's asking about the span. Don't let the vocabulary throw you.

Definition (II.5.20): for $A \subseteq V$, the linear hull $\langle A \rangle$ is the subspace of all finite linear combinations of elements of $A$ — equivalently, the smallest subspace containing $A$. Two equivalent definitions:

  1. Bottom-up: all finite linear combinations $\lambda_1 a_1 + \dots + \lambda_k a_k$ of elements of $A$.
  2. Top-down: the smallest subspace of $V$ containing $A$ (the intersection of all subspaces containing $A$).

Special cases: $\langle \emptyset \rangle = \{0\}$, and $\langle W \rangle = W$ for a subspace $W$. If $A \subseteq B$ then $\langle A \rangle \subseteq \langle B \rangle$.

In $\mathbb{R}^3$:

  • 1 nonzero vector → line through origin.
  • 2 non-parallel vectors → plane through origin.
  • 3 non-coplanar vectors → all of $\mathbb{R}^3$.

Adding parallel/redundant vectors does not grow the span/hull.

Basis

A linearly independent spanning set. Three equivalent characterizations:

  1. L.i. AND spanning (definition).
  2. Minimal spanning set.
  3. Maximal linearly independent set.

Unique-representation property: every $v \in V$ has exactly one representation as a linear combination of basis elements. This is what makes coordinates possible.

The basis machinery used in every big proof
  • Basis extension (Cor III.1.48(ii)): every l.i. set extends to a basis. Used in Rank-Nullity, Dim Theorem for Sums.
  • Basis containment (Cor III.1.48(i)): every spanning set contains a basis.
  • All bases have the same size (Steinitz): defines dimension.

Other terms that may appear (don't get caught out)

  • Standard / canonical basis: the basis $\{e_1, \dots, e_n\}$ of $\mathbb{F}^n$ ($e_i$ has a 1 in slot $i$, 0 elsewhere). "Canonical" = "standard" here.
  • Isomorphism: a bijective linear map. Two spaces are isomorphic if there's an isomorphism between them. Key fact: two finite-dimensional spaces over $\mathbb{F}$ are isomorphic $\iff$ they have the same dimension. (So $\mathbb{R}_3[X] \cong \mathbb{R}^4 \cong M_2(\mathbb{R})$, all dimension 4.)
  • Homomorphism: in this course, a synonym for linear map (a vector-space homomorphism). $L(V,W)$ is the space of all linear maps $V \to W$.
  • Coordinates / coordinate vector $[v]_B$: the scalars $(c_1,\dots,c_n)$ with $v = c_1 b_1 + \dots + c_n b_n$ for a basis $B = \{b_1,\dots,b_n\}$. Unique by the basis property.

Change of basis

Re-expressing the same vector in a different basis. For a basis $B = \{b_1,\dots,b_n\}$ of $\mathbb{F}^n$ and the standard basis $S$:

  • $[\mathrm{id}]_B^S$ (from $B$-coords to $S$-coords) is the matrix whose columns are the basis vectors $b_1,\dots,b_n$. Call it $B$. So $[v]_S = B\,[v]_B$.
  • $[\mathrm{id}]_S^B$ (from $S$-coords to $B$-coords) is its inverse $B^{-1}$. So $[v]_B = B^{-1}[v]_S$.
  • Self-check: $[\mathrm{id}]_B^S \cdot [\mathrm{id}]_S^B = I$.

The notation $[\mathrm{id}]_{\text{from}}^{\text{to}}$ reads bottom→top: takes coordinates in the bottom basis, returns them in the top. The matrix whose columns are the basis vectors is always the "$B \to S$" one.

To do it: (1) stack the basis vectors as columns to get $B = [\mathrm{id}]_B^S$; (2) invert (augment $[B \mid I]$, row-reduce — over $\mathbb{Z}_p$, mod $p$) to get $[\mathrm{id}]_S^B = B^{-1}$; (3) verify $BB^{-1} = I$. To prove $B$ is a basis: row-reduce to $n$ pivots (i.e. $B$ invertible).

Linear independence

$A = \{v_1, \dots, v_m\}$ is linearly independent iff the only linear combination equalling $0$ is the trivial one:

$$\lambda_1 v_1 + \dots + \lambda_m v_m = 0 \quad\Longrightarrow\quad \lambda_1 = \dots = \lambda_m = 0.$$

If some nontrivial combination (not all coefficients zero) gives $0$, the set is linearly dependent.

Three equivalent views (III.1.36):

  1. No nontrivial relation $\sum \lambda_i v_i = 0$.
  2. Every $v \in \langle A \rangle$ has a unique representation.
  3. Removing any element strictly shrinks the span.

How to prove a set is linearly independent

The definition is an implication ("if a combination is $0$, then the coefficients are $0$"), so the proof always has the same shape:

  1. Assume a general combination equals zero: write $\lambda_1 v_1 + \dots + \lambda_m v_m = 0$ with unknown scalars $\lambda_i$.
  2. Solve for the coefficients. Turn the vector equation into conditions on the $\lambda_i$ — usually by comparing components (in $\mathbb{F}^n$), comparing coefficients (for polynomials/matrices), or applying a linear map.
  3. Conclude that every $\lambda_i = 0$. That — and only that — proves independence. $\square$
Why "= 0"?

You assume the combination equals $0$ because that's the one case the definition constrains. Zero is the only vector an independent set must reach in just one (trivial) way; a combination equalling any other vector says nothing about independence.

Concrete methods by setting

  • In $\mathbb{F}^n$ (column vectors): stack the vectors as columns of a matrix and row-reduce. Independent $\iff$ a pivot in every column $\iff$ only solution to $A\lambda = 0$ is $\lambda = 0$. (This is the Problem 6 "basis containment" check.)
  • Polynomials / matrices: set the combination equal to the zero polynomial / zero matrix and compare coefficients (or entries) degree-by-degree; each comparison forces one $\lambda_i = 0$.
  • Abstract spaces: apply a convenient linear map or evaluation. E.g. for functions, plug in well-chosen inputs to peel off one coefficient at a time.

Worked example (polynomial style)

Show $\{1,\ 1+X,\ 1+X+X^2\}$ is independent in $\mathbb{F}_2[X]$. Assume

$$\lambda_1(1) + \lambda_2(1+X) + \lambda_3(1+X+X^2) = 0.$$

Collect by power of $X$: $(\lambda_1 + \lambda_2 + \lambda_3) + (\lambda_2 + \lambda_3)X + \lambda_3 X^2 = 0$. The zero polynomial has all coefficients zero, so

$$\lambda_3 = 0,\quad \lambda_2 + \lambda_3 = 0 \Rightarrow \lambda_2 = 0,\quad \lambda_1 + \lambda_2 + \lambda_3 = 0 \Rightarrow \lambda_1 = 0.$$

All coefficients vanish, so the set is independent. $\square$ (Notice the back-substitution — solve top-down from the highest power.)

To prove dependence instead

You only need to exhibit one nontrivial relation — find scalars, not all zero, with $\sum \lambda_i v_i = 0$. A single explicit witness suffices; you don't need to show all relations.

Quick disqualification: more vectors than $\dim V$ ⇒ linearly dependent, by Steinitz. Used constantly on Problem 6.

The mistake everyone makes

"No vector is a scalar multiple of another" is not the definition and is wrong for sets of 3+ vectors. Three vectors can be pairwise non-proportional yet still dependent (e.g. coplanar vectors in $\mathbb{R}^3$). Always use the formal "only trivial combination gives $0$" definition.

Part VII · Problem 6 patterns

The mixed-bag toolkit

Problem 6 has four subparts. Each maps to a recurring pattern. Recognize the pattern, apply the technique, cite the result, move on.

Fixed-point subspace

Question: "Show that $H = \{v : h(v) = v\}$ is a subspace."

Two-line solution: $h(v) = v \Leftrightarrow (h - \text{id})(v) = 0 \Leftrightarrow v \in \ker(h - \text{id})$. Since $h$ is linear, so is $h - \text{id}$. By Prop III.2.11(ii), $\ker(h - \text{id})$ is a subspace. $\square$

Polynomial subspaces (vanishing, derivative, integral)

Pattern: $\{p \in \mathbb{F}[X] : \phi(p) = 0\}$ for some linear $\phi$.

Examples of linear $\phi$: $p \mapsto p(a)$ (evaluation), $p \mapsto p'(a)$ (derivative), $p \mapsto \int_a^b p$ (integration).

Solution: Express the set as $\ker(\phi)$. Cite Prop III.2.11(ii). $\square$

Dimension arguments via Steinitz

Pattern: "Does $V$ contain a linearly independent (or spanning) set of size $k$?"

Question typeAnswerReason
L.i. set of size $k$? Yes iff $k \leq \dim V$ By Steinitz, l.i. ⇒ $|A| \leq \dim V$
Spanning set of size $k$? Yes iff $k \geq \dim V$ By Steinitz (part ii), spanning ⇒ $|A| \geq \dim V$

Always cite Steinitz (III.1.47) explicitly.

Key dimensions to memorize: $\dim \mathbb{F}^n = n$, $\dim \mathbb{F}_n[X] = n+1$, $\dim M_{m \times n}(\mathbb{F}) = mn$.

Direct sum tricks ($f^n = f$ pattern)

Pattern: "Given $f^5 = f$, show $V = \text{im}(f) \oplus \ker(f)$."

The trick: use $f^{n-1}$ as a projection. For $f^5 = f$, use $f^4$.

  1. Decomposition exists. For any $v$, write $v = f^4(v) + (v - f^4(v))$.
    • $f^4(v) \in \text{im}(f)$ since $f^4(v) = f(f^3(v))$.
    • $v - f^4(v) \in \ker(f)$ since $f(v - f^4(v)) = f(v) - f^5(v) = f(v) - f(v) = 0$. ✓
  2. Intersection trivial. Suppose $w \in \text{im}(f) \cap \ker(f)$. Then $w = f(u)$ for some $u$, and $f(w) = 0$. Apply $f^4$ to $w = f(u)$: $f^4(w) = f^5(u) = f(u) = w$. But $f(w) = 0$ ⇒ $f^k(w) = 0$ for $k \geq 1$. So $w = f^4(w) = 0$. ✓ $\square$

Maps over $\mathbb{Z}_p$

Pattern: "Given $f: \mathbb{Z}_p^n \to \mathbb{Z}_p^m$, is it injective/surjective?"

  1. Write the matrix of $f$.
  2. Row-reduce over $\mathbb{Z}_p$ (mod $p$ arithmetic — careful!).
  3. Read off rank.
  4. Apply rank-nullity: $\dim V = \text{rk}(f) + \dim(\ker f)$.
  5. Injective ⇔ $\text{rk}(f) = \dim V$. Surjective ⇔ $\text{rk}(f) = \dim W$.

Quick disqualifiers: $\dim V > \dim W$ ⇒ never injective. $\dim V < \dim W$ ⇒ never surjective.

Basis extension

Pattern: "Does there exist a basis of $V$ containing the vectors $v_1, v_2$?"

  1. Form $[v_1 \mid v_2]$ and row-reduce to check linear independence.
  2. If l.i.: yes, cite Corollary III.1.48(ii) (basis extension).
  3. If l.d.: no, cite III.1.36 (l.d. set can't be in any basis).

Complex kernel + injectivity ($T:\mathbb{C}^2 \to \mathbb{C}^2$)

This is the most reliable computational sub-question

Problem 6(d) was this exact pattern on both recent exams (3 points each time): a $2\times 2$ complex matrix $T(z,w) = \big(az + bw,\ cz + dw\big)$, "compute the kernel, is $T$ injective?" Expect it again.

The shortcut — don't row-reduce unless you have to. For a $2\times 2$ map, $T$ is injective $\iff \ker T = \{0\} \iff$ the two columns are linearly independent $\iff ad - bc \neq 0$ (the $2\times 2$ "cross-product" test — this is just whether one column is a complex multiple of the other; it is not a determinant method on the syllabus, just the rank-1 check).

Why $ad = bc$ is the proportionality test (click to expand)

The homogeneous system is $az + bw = 0$ (row 1) and $cz + dw = 0$ (row 2). Two cases:

  • Rows point different directions (not proportional): the two equations constrain $(z,w)$ independently, and only $(0,0)$ satisfies both. Kernel zero → injective.
  • Rows proportional (same direction): row 2 is just a rescaled row 1, so you effectively have one equation in two unknowns — infinitely many solutions forming a line. Kernel 1-dimensional → not injective.

So injectivity reduces to: are the rows proportional? The algebra: rows $(a,b)$ and $(c,d)$ are proportional iff $(c,d) = \lambda(a,b)$, i.e. $c = \lambda a$, $d = \lambda b$. Then $ad = a(\lambda b) = \lambda ab$ and $bc = b(\lambda a) = \lambda ab$, so $ad = bc$. Conversely $ad - bc \neq 0$ means no such $\lambda$ — rows independent, kernel $\{0\}$.

In one line: $ad - bc = 0 \iff$ rows proportional $\iff$ rank 1 $\iff$ kernel is a line $\iff$ not injective.

Exam wording: $ad-bc$ is the determinant, but determinants are out of scope in this course — don't cite "det $\neq 0 \Rightarrow$ invertible." Say instead "the two rows are linearly dependent, so the system reduces to one equation." Same computation, course-appropriate justification. (Equivalently: just row-reduce — subtract a multiple of row 1 from row 2 and see if row 2 vanishes.)

Why $\ker T = \{0\}$ means injective (click to expand)

The one move that makes it work: for a linear map, $f(u) - f(v) = f(u - v)$. So "two outputs equal" $\iff$ "their difference maps to $0$" $\iff$ "$u-v \in \ker f$."

($\ker f = \{0\} \Rightarrow$ injective): if $f(u) = f(v)$ then $f(u-v) = f(u)-f(v) = 0$, so $u - v \in \ker f = \{0\}$, giving $u - v = 0$, i.e. $u = v$.

(injective $\Rightarrow \ker f = \{0\}$): if $x \in \ker f$ then $f(x) = 0 = f(0)$, and injectivity forces $x = 0$.

Intuition: the kernel measures how much gets crushed to $0$. A linear map can only merge two distinct points $u \neq v$ if it crushes the nonzero direction $u - v$ to zero. Nothing crushed (except $0$) ⇒ nothing merged ⇒ one-to-one.

Why it needs linearity: for a general function this fails — $g(x) = x^2$ has only $0 \mapsto 0$ yet isn't injective ($g(2)=g(-2)$). The equivalence lives entirely on $f(u-v) = f(u)-f(v)$.

  1. Write the matrix $T = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$ from $T(z,w) = (az+bw,\ cz+dw)$.
  2. Check if rows/columns are proportional. Compute $ad$ and $bc$.
    • If $ad \neq bc$: the map has rank 2, $\ker T = \{0\}$, $T$ is injective. Done.
    • If $ad = bc$: rank is 1 (assuming not all zero), kernel is 1-dimensional, $T$ is not injective — proceed to find it.
  3. To find a nonzero kernel vector when $ad = bc$: solve $az + bw = 0$, i.e. $z = -\frac{b}{a} w$. Set $w = a$ (clears the denominator): kernel is spanned by $\begin{pmatrix} -b \\ a \end{pmatrix}$. So $\ker T = \left\langle \begin{pmatrix} -b \\ a \end{pmatrix} \right\rangle$.

Complex arithmetic reminders (where points get lost):

  • $i^2 = -1$. To divide, multiply by the conjugate: $\dfrac{1}{a+bi} = \dfrac{a-bi}{a^2+b^2}$.
  • Two columns $\binom{a}{c}, \binom{b}{d}$ are dependent over $\mathbb{C}$ iff $ad = bc$ as complex numbers — carry the $i$ terms carefully through the multiplication.
  • $\dim_{\mathbb{C}} \mathbb{C}^2 = 2$ (as a $\mathbb{C}$-vector space). Rank-nullity: $2 = \dim(\text{im}) + \dim(\ker)$.

Worked check (exam 1 style): $T(z,w) = \big((1+5i)z + 2w,\ (8+i)z + (1-3i)w\big)$. Here $ad = (1+5i)(1-3i) = 1 - 3i + 5i -15i^2 = 1 + 2i + 15 = 16 + 2i$, and $bc = 2(8+i) = 16 + 2i$. Since $ad = bc$, $T$ is not injective; kernel spanned by $\binom{-2}{\,1+5i}$. (Always recompute the arithmetic on the day — the point is the method.)

Is this map linear?

Pattern: "Is $F: \mathbb{R}^n \to \mathbb{R}^m$ given by [formula] linear?" (exam 2, Problem 6(c), 1 point.)

A map is linear iff every output component is a homogeneous linear combination of the inputs (each term is constant × one variable, first degree, no constant term, no products, no powers).

  • Scan each component for a non-linear term: anything like $v_i^2$, $v_i v_j$, $\sin v_i$, $|v_i|$, or a constant $+c$ breaks linearity.
  • If non-linear: say "no" and give a one-line counterexample to additivity or homogeneity. For a squared term $v_1^2$: take $F(2v) \neq 2F(v)$. E.g. with $v = (1,0,0)$, the offending component gives $F(2v) = 4$ but $2F(v) = 2$, so homogeneity fails. $\square$
  • If linear: say "yes", write the matrix $A$ with $F(v) = Av$, and note linearity follows since matrix multiplication is linear.

Image/preimage of a set — the 1-point definitions

Problem 2(a) opens with one of these (1 point) — write them precisely:

  • Direct image of $A \subseteq X$: $\ f(A) := \{ f(a) : a \in A \} \subseteq Y$.
  • Preimage of $B \subseteq Y$: $\ f^{-1}(B) := \{ x \in X : f(x) \in B \} \subseteq X$. (Note: defined even when $f$ is not invertible.)
